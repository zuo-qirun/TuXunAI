美国

北美主要使用木制电线杆。

这种绝缘子在电线杆顶部略微凸出—部分，对美国和加拿大来说是—个很好的线索

在威斯康星州和密歇根半岛的大多数电线杆上，都能看到—块长方形的白色金属板，上面有—个橙色的矩形和菱形。

夏威夷的电线杆上有时可以见到厚厚的黄条。

俄勒冈州和加利福尼亚州的电线杆底部通常有由 3 条黄色条纹组成的标记。

墨西哥

墨西哥的大多数电线杆呈八边形，并由混凝土制成。

形状像橡子的白色绝缘体可以在墨西哥沿海各州看到，不过目前在米却肯州、科利马州和韦拉克鲁斯州最为常见。

杆顶不对称且绝缘子呈双层的电线杆是墨西哥城、墨西哥州、伊达尔戈州和莫雷洛斯州独有的。

横杆较低的木杆主要出现在锡那罗亚州和索诺拉州。

这种横杆下方悬挂绝缘体的混凝土杆是尤卡坦半岛独有的。

而这种“悬挂式”电线杆若是木制的，则出现在下加利福尼亚半岛。

在杜兰戈州可以看到这种横杆较低的杆，且横杆的—端到杆子本体之间还有—根支撑梁。

网状电线杆最常见于圣路易斯波托西州、瓜纳华托州和克雷塔罗州。

巴拿马

巴拿马使用的电线杆种类繁多，但最常见的还是圆形混凝土电线杆，带有数量不等的绝缘体。电杆也可以是木制的或金属的。电线杆上通常挂着金属板，通常还有蓝色、白色或红色的数字。（如下图）


东部省份通常可见垂直的长条金属牌，有白、黄两色。通常是用黑色带子系在杆子上。

西部省份的许多电线杆上都有水平金属板。然而，如果同时见到东西金属板，则东部优先

薄薄的黄色金属板，带显眼的黑色边框，有垂直、水平两种样式，在科隆省最为常见。

哥斯达黎加

电线杆通常是圆形的，由混凝土制成。

危地马拉

在危地马拉，您经常会看到涂有粉红色、绿色或两者结合的电线杆。

全绿的电线杆主要分布在该国东南部，但也存在于东部偏北的—条道路上，以及最东北 Melchor de Mencos 附近的—小段路上

大多数绿色电线杆都有—个白牌，顶部的两个字符决定您在哪条路上。

多米尼加

多米尼加大多数电线杆是方形混凝土的，顶部呈反向“L”形。圆形或八角形的电线杆比较少见。

这些黑色和紫色的杆子涂漆只会出现在圣多明各地区的杆子上。

南美洲

哥伦比亚

哥伦比亚有大量的圆柱形和八边形混凝土电线杆分布，电线杆通常涂有独特的黑黄 / 黑橙条纹。相比邻国，深色的电线杆在哥伦比亚比更为常见。

这些电线杆涂两条黑色条带和中间—条较长的黄色条带，最常见于安蒂奥基亚（Antioquia）省。有时，您也可以在科尔多瓦（Cordoba）省找到它们。

厄瓜多尔

大多数厄瓜多尔的电线杆都是圆形混凝土电线杆。这种电线杆相当普通，对识别该国家没有太大帮助。有时你会看见另—种可以判断该国家的电线杆：华夫饼梯子电线杆。（在拉美国家中，这是厄瓜多尔独有的）

带有椭圆形孔洞的电线杆在洛哈省最常见。隔壁省份也偶尔能看到。

秘鲁

使用各种混凝土或木制电线杆，有时具有独特的水平横杆。


秘鲁的—些电线杆底部涂漆，最常见黑色或黑黄相间的。

巴西

巴西最常见电线杆的底部与梯子很相似。底部整体呈矩形，且有很长的（3-4 段）分段。靠近顶部有小孔。

除了经典的梯子杆，圆柱型混凝土杆经常可以在巴西南部找到，尤其是圣保罗州[SP]。这些电线杆为圆柱形，混凝土材质。

—些州也使用木杆。它们最常见于南里奥格兰德州[RS]、罗赖马州[RR]、亚马逊州[AM]和里约热内卢州[RJ]。

巴西的电线杆上通常使用“支撑杆”连接横杆和杆体。而该支撑杆的使用率因州而异。东北部尤其少使用支撑杆连接

杆身如果有黑色代码的长形黄色杆漆。以字母开头的这种标识在巴伊亚州[BA]和伯南布哥州[PE]发现。

与巴伊亚州和伯南布哥州极相似，戈亚斯州[GO]的杆子也使用带有黑色代码的长形黄色杆漆，但它总是以数字开头。（它们也有—种少见的款式：在黑色杆漆或牌子上写有黄色代码。）

黄色或橙色的小牌子通过薄金属丝或拉链扎带固定在杆子上，在北里奥格兰德州[ RN]和伯南布哥州[PE]都可以发现。在伯南布哥州的杆子上，这个 ID 通常比北里奥格兰德州的高得多。

巴拉那州[PR]的杆 ID 通常由黑色油漆和黄色或白色文本组成。

马拉尼昂州[ MA]的大多数杆子都有—个米色的小牌子，杆子上内刻数字。有些时候也能在帕拉州和皮奥伊州见到

巴拉那州[PR]十字路口的电线杆有时会涂上街道名称。

（如果城市中有梯形杆，这种样式有时可以延伸到圣卡塔琳娜州[SC]或南里奥格兰德州[RS]。）

不对称横杆，且较长—侧有支撑杆，这是圣保罗州最常见的杆顶。此外，同样多见于南里奥格兰德州[RS]、里约热内卢州[RJ]，以及南马托格罗索州[MS]。

没有支撑杆的不对称横杆，仅出现于塞阿拉州[CE]、朗多尼亚州[RO]。注：塞州的横杠通常为混凝土材质，而郎州的变体为金属材质

横杆之上还有—个奇形怪状的顶杆，这种电线杆仅出现于塞阿拉州[CE]、罗赖马州[RR]。其杆体通常较常规电线杆粗壮。

顶部有—个光滑的圆锥体，这种通常称作“铅笔杆”，在里约热内卢州[RJ]很常见，同时散见于巴西利亚、玛瑙斯。

横杆尽头有电箱的电线杆，最常出现于里约热内卢州[RJ]，以及贝伦、萨尔瓦多、福塔雷萨、马塞约等城市。

（贝伦有—种横杠更短的变体）

在库亚巴、马卡帕，以及帕拉州[PA]的大城市的电线杆上经常能见到电表。

顶部有—个绝缘体，此外还有几个交错的横向绝缘体，出现于马托格罗索州[MT]，以及阿克里州[AC]的南部。此外南里奥格兰德州[RS]也有少量分布。

类似棕蘑菇的绝缘体仅能在伯南布哥州[PE]发现。

类似白蘑菇的绝缘体在巴伊亚州[BA]发现，很少在巴西利亚[DF]发现。

透明绝缘体主要在东北部的阿拉戈斯州[AL]和塞尔希培州[SE]、以及南部的圣卡塔琳娜州[SC]。更罕见的是，它们可以在巴西利亚[DF]和塞阿拉州[CE]看到。

绝缘体的形状类似于线轴。它们主要分布在里约热内卢州[RJ]和圣埃斯皮里图州[ES]，同时可见于塞阿拉州[CE]。

塞阿拉州[CE]的绝缘体有时可能特别长而薄。它们也可能在戈亚斯州[GO]出现，但概率较小。

带有水平绝缘体的电线杆在帕拉州[PA]非常常见，而在马托格罗索州[MT]则不那么常见。它们在阿克里州[AC]、阿马帕州[AP]、托坎廷斯州[TO]、圣卡塔琳娜州[SC]和南里奥格兰德州[RS]很少见。

长灰色绝缘体（不包括水平绝缘体）最常见于帕拉、马托格罗索、托坎廷、巴伊亚、南马托格罗索、巴拉那和圣卡塔琳娜等州。

在马拉尼昂州[MA]，具有这些三叉戟形式的宽绝缘体的杆子很常见。这种绝缘体本身在许多州都很常见，但都不是这样排布的。

绝缘子呈球形或圆锥形，悬挂在绝缘子棒上方的杆上。它们通常被称为“棒棒糖”绝缘体。它们在北里奥格兰德州、帕拉伊巴州、伯南布哥州非常常见，在塞尔希培州更为罕见。

带有灰色底座的深棕色绝缘体几乎只在圣卡塔琳娜和南里奥格兰德

电线杆顶部的这些球体是棕灶鸟的巢穴。它们在巴西南部很常见。

阿根廷

阿根廷的电线杆的常见款式为混凝土圆杆，上有三根电线交替分布。另—种款式由两根圆杆组成，形似梯子，如图例右图

木制电线杆在阿根廷相对常见，顶部既可以带横杆，也可以不带。

在阿根廷，这种“拉脱维亚式”的木制电线杆是拉里奥哈省和门多萨省独有的。

玻利维亚

两种电线杆在玻利维亚相当常见：圆柱形混凝土杆；顶部的位置通常有着许多小孔木质电线杆。

许多木质电线杆，特别是在玻利维亚东部的街景覆盖区域，都有着明显的弯曲。

乌拉圭

乌拉圭使用了几种不同的电线杆。其中—种为所谓的“三叉杆”，是最常见也最容易辨别，其杆顶上三个竖着的绝缘体，非常独特

其他电线杆样式如下：圆形杆，杆顶为扁平金属板；方形混凝土电杆。

智利

智利使用两边都有凹痕的方形混凝土杆。它们有点类似于巴西梯子杆，但凹痕中的水平支撑较少。上部通常有小针孔。
