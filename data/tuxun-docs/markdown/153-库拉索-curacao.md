# 第 1 步 – 识别库拉索

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912452015-aea98d21-b8dc-42ef-bf12-8062a33b508a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912452979-0d01bade-ee7c-466b-8a14-f9439f83705e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

库拉索是荷兰王国的一个组成国，位于加勒比海上、委内瑞拉西部以北。国家连胜中算作荷兰。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912453627-b8a961de-3fc2-4c47-9316-27738fa03e1a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

库拉索岛使用方形车牌，模糊时全白。

虽然要求挂前车牌，但少数仍然没有安装。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912455146-17fa14c8-31c2-4278-9234-ef7a6d2e4939.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

库拉索为半干旱气候，仙人掌很常见。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912454323-43ceefb8-79ef-4832-a0ac-86dcbad7ddca.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

库拉索的大部分建筑为简单的单层平房，色彩柔和。金属屋顶在较小的棚屋中常见，而瓷砖屋顶则可以在更昂贵的房屋中看到。此外，有时也可以看到多层别墅。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912455185-c8fecaf9-0ce3-4ca5-b882-8d6c043d67c4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在城中可以看到黑白相间条纹的信号灯柱。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912456323-cfafefbd-64a7-4926-8492-782eaf094c56.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

带有三个水平交替绝缘子的木质电线杆很常见。

这种杆顶也可以在巴西、南非和斯威士兰找到。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912458107-fb76ab2b-dd02-4b71-a45e-14d1052ee946.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这辆带有行李架的黑色汽车是库拉索独有的。

# 第 2 步 – 区域线索

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912458505-c6adb7d5-ea45-4c46-99be-82bdb308667d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

库拉索岛的西半部为半干旱的乡村景观，有绿色但干燥的丘陵。房屋分散，道路很窄。

# 第 3 步 – 聚焦

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912458674-50a261cf-7265-43da-979d-af4e601892c6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在库拉索岛东南部的风电场有一排风力涡轮机，可见明显雨意。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698912463027-c276d26a-0b5b-4f92-b96b-8df2496a8a60.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

朱莉安娜女王大桥是威廉斯塔德的一座宽阔的四车道桥。它可以通过黄蓝两色的护栏+城市景观来识别。
