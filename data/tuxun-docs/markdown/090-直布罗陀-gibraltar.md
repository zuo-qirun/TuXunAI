# 第一步 - 识别直布罗陀

![summary_gibraltar.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707098403-d46a3773-b436-4c17-bac0-d6992f99b69d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

直布罗陀是英国海外领土，位于伊比利亚半岛南端，与西班牙接壤。出于国家连胜目的，该领土被视为英国（在图寻中）。

![Gibraltar_antenna.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707192638-35ae59d5-1c23-458b-86db-31c1bccad982.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

直布罗陀的所有覆盖是第三代。车上总会有一个独特的长天线，顶端为银色；社区中通常简称为 "直布罗陀天线"。

在欧洲，除了直布罗陀，这种天线仅在西班牙的部分地区（并不与直布罗陀接壤）还能被找到。

![Rock_of_Gibraltar.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707309571-36cdc9be-a0f8-4dc2-9b68-a9d956f378a5.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

直布罗陀巨岩是此地区最容易识别的特征，它是一座巨大的石灰岩山，在直布罗陀的每个地方都基本可以望见，可通过裸露的白色石灰岩和独特的形状来识别。

![Back_plates.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707368645-ab3caf46-d68e-418c-93d8-8535358db994.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

与英国一样，直布罗陀的车辆也使用黄色后牌。

![Driving_side.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707404067-dd2b015f-bdfc-48d2-99bf-ef607838d3b1.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

但请注意，与英国不同的是，直布罗陀靠右行驶。

![English_language.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707419639-222dd76f-79ed-4417-997b-6d46ff18b0fd.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

直布罗陀主要使用英语。如果你在地中海环境中看到官方标志上的英语，这可能很有帮助。

请注意，英语也是马耳他的主要语言之一，马耳他也有地中海的风景和氛围。

![Gibraltar_architecture.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707470453-11a2df0d-cdbf-4393-a81a-02d40e163108.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

直布罗陀城区的房屋相当密集。建筑物看起来与其他地中海国家的建筑物相似：它们有柔和的色彩和百叶窗——这些百叶窗通常被涂成绿色、蓝色或棕色。

![Christmas_decorations2.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707477463-605a89d2-f9c1-4c45-a18d-ee0fee898ce3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

所有的街景均在12月份拍摄，因此您可能在城市中看到许多圣诞装饰。

![gi_monke.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707553825-140865bf-c307-4038-9c9e-191ccc4216a4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

直布罗陀因其猴子而闻名，属于巴巴里猕猴种。它是欧洲唯一可以找到野生猴子的地方。

# 第二步 - 聚焦

![Mosque.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686707612974-da471898-811f-4d4a-8409-26c92111b2ba.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在领土的南端附近，你将能够看到易卜拉欣-易卜拉欣清真寺的尖塔，这是欧洲大陆最南端的清真寺。

译者注：易卜拉欣-易卜拉欣清真寺这个名字确实不是重复的。
