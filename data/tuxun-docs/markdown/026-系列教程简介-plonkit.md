# 系列教程简介

来源：https://www.yuque.com/chaofun/tuxun/plonkit

Plonk ItPlonk It 是一个不断壮大的群体，其中的每一员都对全景猜测类地理游戏抱有极大的热情。无论作为新人，一个内容创作者，还是一个经验丰富的玩家，你都会在社区中找到一个友善和包容的空间。

Plonk It 指南 旨在帮助几乎所有水平的玩家继续提高。在每篇文章里，你会看到很多图片线索，并附有有用的描述。图片旁边的提示会给你更详细的解释，抑或与其他国家或地区进行比较。

需要注意的是，此系列指南只包括有谷歌官方街景的国家和地区。

作为一个分步指南，每一篇文章都由三个部分组成：

1. 辨认国家：这部分教导用户如何通过一些线索来辨认国家，例如国家特征、相机特点、建筑风格等。

2. 地区和细分线索：这部分提供了一些更具体的线索，帮助用户缩小范围，使用一些技巧确定国家的具体地区或行政区划。

3. 聚焦重点：这部分提供了一些非常具体的线索，帮助用户更准确地选择国家内的某一特定地域。这些线索可能包括特定的地标、道路特色等。

不同学习阶段的图寻玩家通常具备不同水平的知识储备。因此，您非常有必要了解如何高效学习，以便节省学习时间，并更快地提升自身能力。译者在此为您提供以下建议：

- 如果您刚刚接触图寻游戏，建议首先阅读[谷歌街景覆盖与代数](https://www.yuque.com/chaofun/tuxun/coverage_and_generations)文档。它是地点猜测的基础，详细介绍了谷歌街景的基本覆盖范围，以及街景代数的分类方式。熟悉这些内容，能够有效避免您在各类游戏模式中出现错判大洲或国家的失误。

- 如您期望选对大部分国家，优先通读每篇文档的第一部分是相当重要的，它们系统阐述了初步猜测国家的方法，有助于玩家在竞技模式中提高自身竞争力。

- 当您在某些题目中需要选择更精准的区域时，文档的第二部分能够为具体猜测区域提供线索，助您更有针对性地进行选点，提高答题准确率。

- 在图寻中，学习领土大国的区域猜测线索往往比关注小型国家的同级别线索更具实际意义。仅确定国家还远远不够，您还需掌握在该国范围内选择合适区域的技巧。

- 如果您希望了解一些极为独特的地点，可查阅文档的第三部分。该部分详细列出了一些在游戏中较为罕见的点位及其特征，而熟悉并掌握它们将在对局中获得显著优势。

- 对于经验已经相当丰富的玩家，综合文档-社区汉化目录下提供了一些翻译好的 Geoguessr 教学文档，它们通常具有各 Plonk It 文档不具备的细节，或是某一领域的专项教程。

- 另外，我们在寻友教程-世界目录下列出了一些图寻玩家编写并投稿的教程文档，供其他玩家参阅。需要注意的是，我们不保证这些文档的内容质量和可信程度，因此不推荐接触图寻时间不长的玩家阅读。
