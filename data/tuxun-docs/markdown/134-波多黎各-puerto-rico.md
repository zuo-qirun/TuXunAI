# 第一步 - 识别波多黎各

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768922042090-f27a17ac-e9cf-4fc0-89ff-1831e64da3f9.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_26%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![2.png](https://cdn.nlark.com/yuque/0/2023/png/38520669/1697310515553-906c8cbd-a549-4f92-b422-7215d6273405.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各是美国的海外领地。因此，就国家连胜而言，波多黎各被算作美国。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/52538357/1742190160026-3d5e47da-5760-43cb-93b4-988241b7287f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各用Smallcam，一种四代相机的变体，在波多黎各可以用这种非常宽的打码和这个经常可见的天线辨别。这种相机比普通皮卡车覆盖的相机要低一些，车头的打码也更圆一些。

![3.png](https://cdn.nlark.com/yuque/0/2023/png/38520669/1697310518613-2199331a-543a-4e3d-96ab-9196ebaf9ea2.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在波多黎各发现的谷歌汽车右前方有一根天线。天线顶端有一个小白点。

关岛和北马里亚纳群岛的谷歌汽车右前方也有一根天线。在猜测之前，请务必留意其他线索。

![4.png](https://cdn.nlark.com/yuque/0/2023/png/38520669/1697310518878-3a3ae9b4-4719-43d4-8bcc-ac835d2b0a6a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各全境使用 787 作为地区电话代码。在使用下示格式的电话号码前面可以看到该号码：

(787) XXX-XXXX

虽然不那么常见，但是939的区号也是能被发现的。

![5.png](https://cdn.nlark.com/yuque/0/2023/png/38520669/1697310518873-dc152dd9-3cc9-4d70-9052-218f31c798fb.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在波多黎各各地的标志牌上，常见的是西班牙语而非英语。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768921795539-066e77f2-f55f-41a8-8e4a-6c05981ee9c2.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

沃尔玛等美国连锁店在波多黎各非常普遍。这有助于将波多黎各与邻近的其他拉丁美洲国家和地区区分开来，因为在这些国家和地区，美国商店并不常见。

![7.png](https://cdn.nlark.com/yuque/0/2023/png/38520669/1697310520845-9c28ca0b-6391-4cda-aefc-2a06c11bb7c8.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各不强制要求前车牌。因此，很少看到悬挂前车牌的汽车。这有助于与关岛和北马里亚纳群岛区分。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768921811835-75285177-6ec9-421f-bafb-5b5f0ba69350.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各农村地区很大一部分覆盖范围是在大型分离式公路上。

![9.png](https://cdn.nlark.com/yuque/0/2023/png/38520669/1697310527301-116cbc43-03db-4b5d-bc15-ced2c7774c7f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

以下是波多黎各高速公路盾牌的示例。从左上到右下依次为：一级公路盾牌、一级城市公路盾牌、二级盾牌和三级盾牌。 美国任何其他州或地区都不会有这些盾牌。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768921829478-0f9951cc-59d4-45a3-b761-354e230c9259.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各的限速标志在形状和大小上与美国本土的限速标志相同。主要区别在于，这些标志始终使用西班牙语，而不是英语。

![11.png](https://cdn.nlark.com/yuque/0/2023/png/38520669/1697310526898-224d11f0-2da2-4132-9d84-cca6701a959f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各的停车标志上写着 "PARE"，而不是 "stop"。

“PARE”在整个南美洲和多米尼加都用。

![12.png](https://cdn.nlark.com/yuque/0/2023/png/38520669/1697310526986-a237f855-43eb-4931-87d3-f14e93037150.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各的公里标志高大且是绿色背景，顶部写有 "km "字样，后面用大字竖写着一个数字。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768921886272-5514d579-ac78-4d5a-b601-85bf354de5c5.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

波多黎各的大多数住宅都是混凝土结构，屋顶是平的。住宅前的金属大门非常常见，尤其是在城市地区。

拉美其它地区的房子也有类似的样式。

# 第三步 - 聚焦

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768921931492-6e225244-76f3-4af6-a619-c195d435a74a.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

南部城市庞塞有许多庞塞克里奥尔建筑风格的建筑物，这种建筑风格是维多利亚风格和西班牙复兴风格的结合。建筑物通常由木材和砖石砌成。

译者注：庞塞克里奥尔建筑(Ponce Creole Architecture)是一种特定于波多黎各坡恩塞市（Ponce）的建筑风格，这种建筑风格结合了多种影响，包括西班牙、非洲、原住民塔伊诺人和法国的建筑元素。以下是一些简单的用于识别Ponce Creole Architecture风格的特征：装饰精美的立面，意大利式琉璃瓦，多彩的外墙，铁艺门窗，高天花板，有内院和花园，建筑的结构通常采用木材。不过这个地方一般也做不到，上图的素材也不完全能体现特点，大家就当长见识了（。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768921949819-345c3c26-6a81-4375-945d-4da12294eaab.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

圣胡安老城位于圣胡安市最西北端。这里的街道非常狭窄，建筑色彩斑斓，混合了哥特式、文艺复兴式和巴洛克式建筑风格。此外，该地区的一些街道是用砖块砌成的。
