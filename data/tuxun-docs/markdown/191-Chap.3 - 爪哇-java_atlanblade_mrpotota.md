★ 爪哇岛上的民居多使用爪哇瓦铺设屋顶。爪哇瓦，顾名思义广布于爪哇岛，一般呈现为橙色或棕色，常附有深色的斑块。爪哇瓦分布广泛，主要有几个分布区域：爪哇岛、苏门答腊岛南部、努沙登加拉群岛西部、东南苏拉威西省、南加里曼丹省。[[示例]](https://maps.app.goo.gl/XTtKjmN1iRXXXP7f9)

其中最主要的分布区域还是爪哇岛，但是不要一看到爪哇瓦就选到爪哇岛上。

    爪哇岛人口分布十分密集，尤其是在大城市附近。由于爪哇岛的人口密度远远超过其他大岛，房屋密度相较于其他岛上会更高，使得每户人家所占有的房屋面积减小，故有时在街景中你会有些“拥挤”的感觉。而在其他岛的非城市地区，人口密度较小，大部分房屋都有一个较大的院子，或门前留有大片的空地，使得房屋看上去离道路更远。此技巧更类似于vibe guess，若对此不是很理解可以跳过，因为一般爪哇瓦就够用了。[[示例]](https://maps.app.goo.gl/8bEZ5tixRDdkHzQdA)

★ 人口密度也对爪哇岛的农业造成了影响。在爪哇岛上，农田随处可见，不论是平原上的水田、旱地，还是丘陵上的梯田、果园。为了养活庞大的人口，爪哇大部分土地都已被开垦开发，以供种植作物，而荒地较为少见。其中最常见的作物是水稻，在爪哇东西部都有广泛种植。其他岛上一般不会出现如此规模的大片农田。[[示例]](https://maps.app.goo.gl/wchsEjywU69MiQe8A)

爪哇是唯一分布有大规模柚木种植林的岛屿。此外，零散的野生柚木也常见于努沙登加拉群岛、苏拉威西南部以及加里曼丹东南部。参见植被-柚木。[[示例]](https://maps.app.goo.gl/QnnE4JLdznaGdhGc9)

    ★ 由于中部山脉阻隔，南部来自印度洋的水汽多于爪哇南部形成降雨，造就南部极好的雨热条件。成片的椰子树一般只出现于贯穿爪哇岛的火山链的南部，在爪哇岛北部的平原地区分布较少。[[示例]](https://maps.app.goo.gl/xj4VfyzhDXACLpe99)

    ★ 在爪哇东部的马都拉岛(Pulau Madura)及其东南的Pulau Puteran，红土十分常见。岛上有很多用红泥糊起来的白砖房（有时也会用灰泥），这几乎是此区域独有的。除此之外，许多砖房和木屋也都会将外墙刷成显眼的白色。[[示例]](https://maps.app.goo.gl/PdUaky3hbeh6y8AB8)

![image.png](https://cdn.nlark.com/yuque/0/2023/png/38682438/1694181965303-475ba433-8310-4d14-bdc4-a44f51f758b6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_28%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730533532715-68a8488f-2963-480b-bc4d-7aa83de0e318.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_38%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38682438/1740818853321-d631f983-e019-4ca1-8305-5115f27b1e23.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_41%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

    越往东走，土壤颜色越红，屋舍的墙脚常常就被这种土壤染成了棕红色。Pulau Puteran可通过随处可见的鲜红色土壤辨认。[[示例]](https://maps.app.goo.gl/fjNDmP7HAKuyYedm6)

![image.png](https://cdn.nlark.com/yuque/0/2023/png/38682438/1694181825042-79eabab0-31c1-4f4a-b538-d4cdcf751bff.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_39%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

    类似于努沙登加拉群岛，这里长有大量的椰子树和扇叶糖棕。也正如龙目岛部分所提到的，矩形的块状农田之间有田埂，田埂上长有草、其他作物或小树。不同于龙目岛，马都拉田地上的田埂没有那么分明，而像是在铺设梯田时自然形成的。

    ★ 由于人口密度较低，你很有可能会看到广阔无垠的，大片起伏的缓坡梯田，有几簇民居零散坐落在远处，这是马都拉岛的普遍景观。岛上大部分覆盖为gen4。[[示例]](https://maps.app.goo.gl/s3HXbn6SncXnhc126)

    马都拉岛西部尚且郁郁青青，东部则更为干燥。在马都拉岛东部，有更大的概率能够看到丝葵和扇叶糖棕等植物。碎石块修成的梯田有时会出现在岛东部。[[示例]](https://maps.app.goo.gl/5caqTH23oTwEXhCk9)

相较东部，马都拉岛西部部分地区的房屋更加类似于爪哇岛，且看上去更为整洁。[[示例]](https://maps.app.goo.gl/cbNWw7QEqxWVsnnC7)

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38682438/1752649371918-60da30b7-891a-4821-92f7-91e02d071aba.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_28%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

马都拉岛有不少由木质低脚架架起的小房屋。[[示例]](https://maps.app.goo.gl/cB4Cgmjib2AX1gQc8)

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38682438/1740816372823-6e082baf-5062-4326-b412-92c52683d958.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_35%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

马都拉岛上有很多类似于Limasan的建筑。但与中爪哇的Limasan不同的是，马都拉岛的Limasan的屋脊上的爪哇瓦一般使用水泥黏合连接，而中爪哇的屋脊一般仅仅简单使用爪哇瓦铺成，不使用水泥。[[示例]](https://maps.app.goo.gl/ZbmW2BcsrnETXBgXA)

#### 巴韦安岛

![t.png](https://cdn.nlark.com/yuque/0/2025/png/38682438/1740804176166-2ce406fb-2ff1-48e0-afe0-d4ad0c581ad3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_24%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

    巴韦安岛(Pulau Bawean)位于爪哇岛北部海域，距离爪哇岛北部海岸约115公里，行政上从属于东爪哇省的锦石(Gresik)，最大城镇为南部的桑卡普拉(Sangkapura)。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730532680922-6b239aca-ec10-455e-85b4-1ea81286b45d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_37%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730533036291-17069d52-70f5-4cb5-bd96-ebae5f29e5e5.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_42%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

    由于拍摄季节降水量较少，巴韦安岛多呈现略微干旱的景观。[[示例]](https://maps.app.goo.gl/yp1NhGPqq5JAX3fE9)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730530857568-20bbfb85-859c-45d6-a7b5-03c714b054db.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_26%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730531246179-fc899eea-eff6-4119-9349-76da08ec8d6d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_46%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

    ★ 巴韦安岛的基础建设较好，环岛主路由这种黑色的方砖铺设，道路质量极好。道路中央使用方砖斜向铺设，两侧为单层横向铺设的方砖。也有部分道路使用沥青或水泥铺设。注意这种方砖的铺设方法也可能在东爪哇其他地区出现。巴韦安岛也有部分水泥板路。[[示例]](https://maps.app.goo.gl/KMcxaoUDvuXKwU9k8)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730531568276-7ac97986-a15b-4789-a6f1-9f05c47b5159.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_14%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730531753944-9fb1fcfd-8b48-4774-83cb-fc66aedb53cb.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730532126897-ae39e39c-7e8f-47ed-9880-5b70e72fff35.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_18%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/38682438/1730532956738-6238ca8d-f8fd-41be-a5de-8c0d08771003.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_16%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

    几乎每条道路都设有路灯，即使是野外的小路。常见的路灯样式如图所示，大多有一个135°的折角。[[示例]](https://maps.app.goo.gl/ch48e1GzeszWbrZ26)

    房屋与爪哇岛一同使用爪哇瓦。基础设施与东爪哇省基本无异，然而带杆顶的金属电线杆的出现频率远远大于东爪哇省。[[示例]](https://maps.app.goo.gl/65uRyPA62eVBee1k7)
