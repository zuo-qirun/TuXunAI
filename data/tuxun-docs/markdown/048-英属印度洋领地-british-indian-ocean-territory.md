# 第 1 步 – 确定英属印度洋领地

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492611156-87169fa8-8b1e-4421-ac2f-df8cc02dcb15.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492603163-140a3d7e-faa5-465b-b84b-ac9792c5bcdc.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_71%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

英属印度洋领地是英国在印度洋的海外领土。因此，在国家连胜中，应当视为英国的一部分。

请注意，这里的街景都无法精确定位，大多数社区制作的地图也都没有收录此地。因此，本指南旨在大致介绍此地的街景类型。

![](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492609592-665b41cf-b9d9-49b8-a267-b61d17cbdd0d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

街景由第 3 代徒步旅行者拍摄。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492611997-e3582e9c-f3fc-4f49-ab28-b49a33b6e827.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_20%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

覆盖范围主要分布在领土北半部的小环礁上。

![](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492619285-78876f7d-f155-4d8b-ae8d-62a084ef1dc2.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

大部分街景是在沙滩上拍摄的。海滩上通常会有许多棕榈树。

![](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492629121-dffac8c8-4a58-4e8a-861f-d52f6bac7413.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

沿海棕榈林也占街景的很大一部分，而且几乎全是椰子树。

![](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492642902-8cbc1992-f0a9-436c-baf1-58a87035e668.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

少数环礁上还有红树林。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492636565-15c10b44-a6aa-4801-93d0-6cc1d5122bbf.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

较大些的岛屿上有草地，四周环绕着灌木丛和棕榈树。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492635122-e0af61a9-fdf6-4ee3-a22e-efea2992b6cb.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

旅行者偶尔会走到海里。

# 第 2 步 – 聚焦

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1705492638095-c6c7561d-1194-4716-b8ee-6cb204bd30ac.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

Coin du Mire是一个小岛，拥有深色的岩石海岸和草地，但没有灌木丛或棕榈树。

尼尔森岛（Nelsons）东端的海滩上有一艘白色的中型沉船。
