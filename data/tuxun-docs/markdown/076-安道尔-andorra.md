# 第一部分：辨别安道尔

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318938514-3bf94279-bd58-4eb7-876c-16e2e0391170.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔车牌左侧有个橙色块，全国统一。

欧洲范围内，仅安道尔有这种车牌。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318939082-4dafe90b-f525-4896-abf1-15a702d0d583.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔仅有第二代、第三代街景。所有第三代街景中，都可在谷歌街景车上看到一根中短长度的天线。

迄今为止，第三代街景范围最广。之前拍摄的第二代街景几乎都被第三代取代，仅部分非铺装小路保留。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318939586-a5219616-db4f-41e9-853b-5ef1b23b4359.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

所有第三代街景都是十月拍摄，因此许多落叶树木都已染上秋色。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318940196-b3b95df4-c04e-41f7-9cdd-1674200fc4c9.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔有着大量石制房屋，几乎遍布全国各地。

法国和西班牙境内靠近安道尔的比利牛斯山区同样有这种房屋。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318940697-26df86d5-0209-4ae4-a426-d8295fd548f1.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔的雪杆是木质的，颜色各异。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689408351066-c342bdba-58e3-4de3-9bcf-87a2bd293836.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔的主要语言是加泰罗尼亚语。它是一种与西班牙语和法语关系很近的罗曼语。

如果您不熟悉罗曼语，可能很难区分加泰罗尼亚语和西班牙语。而如果您看到 i而不是 y（英语中的“and”），则它是加泰罗尼亚语。此外，加泰罗尼亚语单词更多以辅音结尾。

加泰罗尼亚语也使用于西班牙东部：加泰罗尼亚，巴伦西亚自治区和巴利阿里群岛。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318941322-5163e24b-4233-452d-bcbb-2547429ad87f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

加泰罗尼亚语中的街道是“carrer”。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318941825-8a75e80b-a7cc-455e-9a96-5481ca53ecb3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔的里程牌上有“CS”、“CG”字样，后接道路编号。

西班牙同样有类似的里程牌。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318942362-48922052-8933-4f3b-a212-1233abbe92d6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔的道路护栏上有黄色反光体。

欧洲范围内，仅安道尔、西班牙和比利时有这种黄色反光体。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318942866-a5d924cb-83b6-4f95-aff5-eb6e0909eda6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_16%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔的主要干道旁偶尔出现路桩，上有橙色反光体。

西班牙同样有此种路桩。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318943491-38652f99-f44a-4a32-8a7b-941b63c39477.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

黄色和绿松石色的交通路标遍布全国。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318944066-d2843303-3c50-4d33-bf39-060b3e8ed27f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斑马线标志有8道条纹。

欧洲范围内，仅安道尔和西班牙为8道条纹。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318944710-d6e525d5-f7ef-41b9-a5c6-97fc6a5a8977.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔有大量雪地中的第二代徒步街景，滑雪度假区尤为多见。

其他国家也有雪地中的徒步街景，但安道尔最多。

# 第二部分：辨别各区域

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318945331-34ba688a-1d68-430e-b9eb-300713a59538.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔城为该国首都，四面高山环绕，很有辨识度。

该城海拔1023米，为欧洲海拔最高的首都。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318945913-55540bd1-2ba3-45ed-8979-31018a28b55c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔城内几乎随处可见 Caldea tower 和摩天轮。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678318946452-d51b1294-2d0e-4d79-98ad-5e175c69f39d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

安道尔东部的卡萨隘口（Pas de la Casa）是一座旅游城市，有着大量的商店和酒店。该城海拔2080米，四周的山上并无树木生长。
