# 科科斯群岛

# 第 1 步 – 识别科科斯（基林）群岛

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914328378-5acc3709-5c3b-45c8-96a7-23d05389a474.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914323850-cd1c79f3-db42-43cb-b722-3b6220984635.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_51%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

科科斯（基林）群岛由两个珊瑚环礁组成。它是澳大利亚的领土，国家连胜中算作澳大利亚。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914317850-0ae23560-b60e-42fb-bf68-ead47237835d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

总体景观由很明亮的白色沙滩、许多棕榈树和修剪过的草坪组成。与[圣诞岛](https://www.yuque.com/chaofun/tuxun/christmas-island)相比，科科斯群岛的森林、丘陵要少得多。

# 第 2 步 – 区域线索

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698913706881-115dd0d6-876f-482f-9543-a88cdf4db69c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

西岛上这辆独特的街景车在右前方有一个通气管，车框的边缘可见。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698913929835-12f6bbab-ddfd-4f88-8d4a-2a1de43c09b9.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

而在Bantam，街景车的白色后缘可见。

街景车也可能完全被模糊。在猜测之前，请务必注意其他线索。

译注：华为地图上没标 Bantam，请注意其位置。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914364468-ae312663-c0ee-48e6-8d9a-364ccef06aec.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

Bantam可以通过交错的铺砖路面来识别。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914383578-0d38a228-f3be-45e4-bb33-4c90ce3c2272.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

科科斯群岛跑道是岛上最宽的铺砌道路，跑道两侧为割过的草，街景中可见有一辆跟车。

# 第 3 步 – 聚焦

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914323202-e1120cf1-04e3-414e-a434-ef187a455f1d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

科科斯群岛沿岸有许多徒步拍摄者。他们大多走在一边大海、一边棕榈的白沙滩上。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914352686-8b8af6a6-8458-4f63-a2e2-7a45dad7d42b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

与其他岛屿相比，方向岛（Direction Island）海滩上的垃圾明显多得多。海岸的某些地方可以看到砖块、聚苯乙烯泡沫塑料和死珊瑚。

此外，还可以看到徒步拍摄者在岛内陆的许多棕榈树之间行走。

译注：同样，华为地图上并未标该岛的名称。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914326925-b5284f8a-6242-48a3-8e39-8267afbc1db8.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

霍斯堡岛（Pulau Luar）的内陆徒步拍摄街景可以通过干燥的黄色草地和更稀疏的植被来识别。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914329561-a963583f-acf6-4435-a92f-251936097d5e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

从西岛到 Bantam 有一艘较多座位的中型船。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914331360-9781a67b-5b43-41c9-8bdd-304ea7b6df71.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这艘后部有黑条的小船可以在北部看到。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698914332734-3beab54a-1bfa-432a-9591-ddc962a99641.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在南部可以看到这艘深灰色的小船，后面坐着一个女人。
