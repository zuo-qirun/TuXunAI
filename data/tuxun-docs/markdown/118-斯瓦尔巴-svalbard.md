# 第一节 - 鉴别斯瓦尔巴

![Svalbard_in_Europe.svg](https://cdn.nlark.com/yuque/0/2023/svg/34598262/1700653810564-ea92c03b-7f4a-46f2-a0f3-7085f8a9f38b.svg)

斯瓦尔巴群岛是北冰洋上的一个群岛，挪威对其拥有主权。在国家连胜模式中，它们被视为挪威。

![svalbard_licenceplate.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700654223598-f5e6cb06-0a1f-4442-b11f-6113a68d7e3e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯瓦尔巴的车牌是黑底黄字。

![image_2.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700654253473-d44e3135-2de9-4d46-89c4-46f6aa3ef156.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这里的总体地貌是荒芜无树的山脉。树木和绿植非常罕见，大部分植被都是苔藓和野草。天空也经常阴云密布，给人一种阴沉沉的感觉。

![svalbard_russian.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700654392404-565d3225-a83f-4297-8834-3f76955c4550.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

有时可以在标志上看到俄语。

# 第二节 - 市镇特点

![Longyearbyen.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655447194-e9664f7e-21df-45df-aa16-f5381e0b7c6f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

朗伊尔城是斯瓦尔巴群岛最大的定居点，城内有许多方方正正的简约建筑。

![Pyramiden.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655451287-eeae7ea8-d5bf-4a3e-b7ee-f696731651bc.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

与朗伊尔城相比，皮拉米登给人的感觉更为荒凉。这里可以看到多层苏维埃风格的公寓楼，但缺少人类活动。顾名思义，皮拉米登的远处可以看到许多金字塔形的山脉。

![截屏2025-05-25 22.49.14.png](https://cdn.nlark.com/yuque/0/2025/png/57046230/1748184646989-d3fd6a5d-a853-4a9f-bf9e-2597d2ec2d6b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_27%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

金字塔形的山脉

![Barentsburg.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655454723-5b02efb9-89de-488b-af4d-c05a2fa5d256.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在三个城镇中，巴伦支堡给人的感觉最为荒凉。在山顶附近可以看到色彩鲜艳的公寓楼。与朗伊尔城和皮拉米登相比，巴伦支堡的天气更加阴沉。

# 第三节 - 地域性线索

![Longyearbyen_Road.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655469150-da82c6ce-b203-486a-91f8-4ffe764da2d6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这条路从朗伊尔城向东南方延伸，道路两旁是荒芜无树的群山，湖泊直通群山。左侧有时可以看到一面黑色的后视镜。

![Airport_Road.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655700048-0e87b610-bc2c-478a-83bf-fa0f8fd0fce2.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在斯瓦尔巴机场南面的一条蜿蜒公路上，可以看到一名Trekker紧紧抓住一辆红色汽车。如果没有这辆车，东北方向的这一排明显的山脉也是一个很好的指示。

![National_Park_Boat.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655715978-9ce6e320-2d4a-4afd-ad36-11d7f6b3fa93.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在斯匹次卑尔根西北国家公园周围可以看到这艘大船。天色向东越来越晚，最终变得漆黑一片。

![Wooden_Boat.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655719563-caca5c5c-0b0d-4a7f-ba6b-1c39c93c8ddf.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在这些地方可以看到这种木甲板的船。

![Small_Red_Boat.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655726085-4430a047-564a-48f0-93a0-0e179f0fabdf.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在西北部可以看到这种红色小船，船上有穿红色夹克的人。

![Isfjorden_Boat.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655730304-a255502f-5109-4d5a-bfeb-3049d7ac3bd3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在巴伦支堡和格鲁曼特间的伊斯夫峡湾，可以看到这种带有黑色甲板的小船。

![Pyramiden_Boat.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1700655733355-b28b2560-8260-4e95-84c4-b302d8a54b0e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在皮拉米登以东可以看到这艘船，舵手室有倾斜的前舱。
