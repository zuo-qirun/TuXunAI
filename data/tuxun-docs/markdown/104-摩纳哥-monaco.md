# 第一步：辨认摩纳哥

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698310948470-092835ff-6edb-4101-9842-6c64a6c34c00.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686711623741-f0fe82da-35e9-489f-9dbb-50b48d1fc974.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_18%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

摩纳哥是一个位于法国里维埃拉地区（地中海沿岸）且仅在意大利边境以西数千米的微型国家。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698311032747-e4493b21-7ed3-4198-9a50-5666106827aa.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

摩纳哥是一个富有且人口稠密的城市国家，颇具地中海风情。其建筑风格与法国南部和意大利北部大体相似，但更为豪华。公寓楼四处可见，并通常以新古典主义风格的细节和淡雅的颜色装点。特别要强调的是，这里没有乡村地区。

![](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686711626244-c3f6a6a4-44b6-47a6-969e-a16598a33e7b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

摩纳哥的车牌很短，并且没有蓝条。

法国的车牌也很常见，但这些车牌通常还有旧式的黄色后牌照（取决于Gen2相机的年代有多老）。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/57046230/1748180676971-77fdcff8-fc42-416b-8a7e-1a313a2d5acb.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

摩纳哥是一个多山的国家，在少数建筑没有那么密集的地区，分布着地中海特色的植被。

![](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686711627746-a9139077-aed4-4958-87f2-669a7fd97acb.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

人工种植的棕榈树十分常见。

![](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686711628521-fd861afa-36d6-4e6b-b78f-aac66e6ae244.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

隧道里有许多覆盖。

![](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686711629610-35d0b239-af96-417a-b0a0-8af22be80db5.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

成排摆放的踏板摩托车在摩纳哥十分常见。

# 第二步：聚焦

![](https://cdn.nlark.com/yuque/0/2023/png/34598262/1686711630426-0efef443-5aa9-4125-a223-6c1a4a9debf6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在港湾中，船上的Gen3 trekker十分常见。这位谷歌员工的棕色头发和墨镜可见。你可以在远端看到摩纳哥的山丘与天际线。周围停泊着许多价格不菲的游艇。

这位trekker在官方竞技模式中十分常见，值得重点记忆。
