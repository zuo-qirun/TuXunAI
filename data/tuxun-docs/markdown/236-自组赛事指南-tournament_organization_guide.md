# 自组赛事指南

来源：https://www.yuque.com/chaofun/tuxun/tournament_organization_guide

申请流程：

- 填写 [https://f.wps.cn/g/1u2vdogu/](https://f.wps.cn/g/1u2vdogu/) 申报奖励。

- 申请参考：

-

[第⼆届海祇杯世界组队赛.pdf](https://www.yuque.com/attachments/yuque/0/2025/pdf/583371/1754802117023-dbf93990-fb67-49a5-936e-7ae5af883c24.pdf)

图寻官方提供：

- 宣传资源（需要申请）：

- 首页（简单）

- QQ群宣传（简单）

- 公众号（较难，高质量赛事）

- 奖励：

- 优胜选手：会员和/或宝石奖励。

- 赛事组成员：会员或宝石奖励。

要求：

- 赛事组有明确可行的计划。

- 赛事组中没有违规用户。

- 赛事结束以后及时更新公布结果（可在原有报名帖更新）。

- 报名帖标注：

- 图寻官方仅提供赛事奖励和宣传支持，不负责和干预比赛过程和结果，比赛由赛事组负责。

- 自组赛事官方支持申请指南：[https://www.yuque.com/chaofun/tuxun/tournament_organization_guide](https://www.yuque.com/chaofun/tuxun/tournament_organization_guide)

报名帖参考：
