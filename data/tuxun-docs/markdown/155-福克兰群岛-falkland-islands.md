# 第一部分——地点

马尔维纳斯群岛（英称“福克兰群岛”）位于阿根廷火地岛省（Tierra del Fuego）以东，国家连胜中视作阿根廷的一部分。

该地的街景覆盖非常有限：仅有三代徒步拍摄街景，且全部位于西部的三个岛屿及周边地区。

    1、Carcass Island

    2、West Point Island

    3、New Island

注：华为地图无标识。建议使用以下图片记忆。

![85P[3%W9Y{}8S0OG~NQN44E.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1698922686593-41d1b169-e07c-4c92-a733-329f3bc30982.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_19%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

因为覆盖太过稀疏，也完全没有城镇街景，因此就没有任何领土识别技巧。因此建议记忆每一处徒步街景的特征。

# 第二部分——徒步街景

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1698922000699-4f757503-ce9e-45cb-8b53-9057e99062e0.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

Carcass Island的徒步街景特点为开阔的平原，大海清晰可见。此外，地面有时也遍布着石头。

注：注意分辨此地与法罗群岛的徒步街景。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1698922000213-fc5b9a68-083a-49e0-bc22-41ddc29aff8f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

West Point Island上也有开阔平原，但长着微黄的草。这里无法轻易看到大海。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1698922000133-ee12de54-a197-43ed-88a4-c67ecbdee149.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

该船上街景摄于West Point Bay，远处有黄色的山丘，旁边还有MS National Geographic Explorer的大船。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1698921999669-5a7411c2-cc36-482d-8543-5cc7aa781811.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

New Island 的沙滩，画面中可见到其他身穿红蓝色夹克的人。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1698922000934-34fd3c61-daef-4c55-a9c8-f84ff25b840a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

New Island 内陆部分有着深棕色土壤。胎印清晰可见。
