# 第一部分：辨别斯威士兰

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859742653-763733d7-e900-49b1-b24f-1d8d9f7226a4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_29%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859743030-0d8c3a62-9f03-4a67-80de-380233281fa9.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰的地貌特点为长满树木的绿色山丘，以及路旁通常呈金黄色的长草。

莱索托的山脉通常光秃秃，也没有这些长草。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859743661-6f69039c-905f-44d9-858e-c846b7bcdc9d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰的电线杆为深棕色木质，设计很简约。绝缘体通常位于电线杆侧面。

在南部非洲地区，若电线杆上的绝缘体为三个，则最有可能为斯威士兰。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859744345-c11aa6c5-8fb7-47e2-b60f-fe04fc42c336.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

MR开头的道路编号是斯威士兰独有的。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859745669-b96631eb-f121-431b-b618-c48f2cc08fde.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰通常使用黄色边线，有时可见三条白色中线。

南非、莱索托和博茨瓦纳也有该风格的线。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859746284-63a865de-79b1-4960-97d8-b48111aa1145.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰的街景都是由无天线的白色街景车拍摄的。

非洲范围内，南非、莱索托和博茨瓦纳同样如此。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859746811-9c859922-9bc7-4fed-8902-7cce30ec1e8c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰的车牌很长，底部是绿色的。南非部分车牌有着相似设计。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859747716-6d748c37-910b-4890-8068-25dab8b5f824.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰路边常见混凝土公交车候车亭。

# 第二部分：辨别具体区域

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859748288-21afa865-c1c9-48bd-9f8b-d47a48b9b9e4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰西部的典型特征为长满金黄色长草的高山草地，以及多山的地貌。

斯威士兰几乎全国都有长草。这里的长草特指该国西部最常见的金黄色野生长草。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859748865-f0a03b1b-cba3-4363-8a75-ebff73a381a1.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

松树林区位于该国西部，具体范围如上图所示。

南非也有类似的林区。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859749447-39aab4cb-1635-4dd9-ac05-258d09ff0797.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

该国中部偏南地区的很独特，其特征为满屏幕的深绿色，以及树木覆盖着的高山。主要位于MR9道路沿线。

如果不理解该地区的独特之处，可以试着比较该国其他地区，那里的草地更多呈金黄色，颜色更浅。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859750240-0cc2f271-bebf-458a-852d-094fb176b5b6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

该国东部的这片区域难得的平坦，山都位于远处。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859750885-b1c6db2b-abec-4ec4-85bc-6fb008d9c8b4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰靠近东部边境的部分地区种植有甘蔗。

# 第三部分：小技巧

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859751427-b1903387-8bbd-4d77-90d7-e8acc13216be.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

恩戈马内（Ngomane）南侧的MR3道路沿线比较平坦，两侧多干燥的长草，相机上还有尘土。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859751962-8000fcf3-eae0-421e-a1ec-843f245c5630.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

由曼齐尼（Manzini）到南非的MR3道路中间有金属和石头材质的分隔栏，很有特点。

这是斯威士兰唯一的分离式公路。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859752459-dae6084d-7ccf-4b6b-bea7-a849f6b8c08d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯威士兰东南部的MR8道路横跨南非边界，其东面有道山脊，很有特色。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859752967-62cbd9cb-52f4-4900-8f91-1b9564bb2e64.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

姆拉武拉自然保护区（Mlawula Nature Reserve）有着狭窄的土路，长长的草，以及总体上的稀树草原景观。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859753562-e2cd3160-2e35-42cf-826e-65e95be1cc15.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

若在斯威士兰见到葡萄牙文，则最有可能位于Lomahasha镇上。

这是因为该地靠近以讲葡萄牙语为主的莫桑比克。
