# 第一部分：辨别美属萨摩亚

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120409487-50b7a3d3-f10b-4c82-ad12-f72d7d07324e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120410145-2701ef9b-105e-48b4-839e-51ba66e1b52f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

美属萨摩亚是太平洋上的一片群岛，位于斐济以东。该地为美国海外领土，因此在国家连胜中视为美国。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120419829-ae68964d-b866-459f-ae88-360c2c9abd25.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

该地总体景观为绿色的热带群山，以及大片棕榈树。小型道路通常为混凝土材质（见下图）。大多数房屋只有一层，采用简单的盒状造型。平面金属板屋顶也很常见。

![841[4Y2NXN[O$$)`_(4LIBE.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120797266-f19ac574-5657-44dd-a332-5a6d0ec0fdf7.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_11%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

# 第二部分：辨别具体岛屿

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120425169-bb2bc66d-9341-4ded-812c-9fc97c19b93a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

主岛图图伊拉岛（Tutuila。即地图上显示“帕果帕果”的岛屿）上面的皮卡街景车，车斗为黑色，稍有些锈蚀。该车所拍摄街景范围最广，因此也最为常见。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120425371-887654a0-ba42-43fc-8250-5b4655a15ac2.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

银灰色皮卡车头，两侧有后视镜。该车在马努阿群岛（Manu'a）和奥洛塞加（Olosega）上都有分布。两岛有桥梁连接。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120425302-5dd34ac2-68d0-4e1d-99f4-91036c186494.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

该街景车位于塔乌岛（Tau），这是街景覆盖的最东边。该车与上条提示的车类似，但它呈有光泽的黑色，可以看到两侧的后视镜。

# 第三部分：其他

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120425328-1e2c0ff0-a9b1-4448-8c1b-3600371b8764.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这座单车道桥梁连接了奥富岛（Ofu）和奥洛塞加岛（Olosega）。它两侧有纤细的护栏，两头都是岛屿，很容易辨认。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35359367/1697120438930-c5135516-97a8-409d-a506-5ff60b0cc7c9.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

奥富机场规模很小，有条混凝土跑道。这里的谷歌街景车车顶有两道橙色条纹。另外，跑道另一侧还有大量棕榈树。
