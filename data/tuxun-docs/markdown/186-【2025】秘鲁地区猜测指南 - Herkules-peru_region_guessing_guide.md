# 大区

秘鲁总共有 25 个大区。 记忆这些大区名称很重要，因为它们会出现在路牌、电线杆甚至大区道路标志上。很多大区名称都很直观，因为它们都以各大区首府命名。

# 电线杆

电线杆贴纸对于猜测秘鲁地区可能是最有帮助的线索之一。它们出现频率很高，可以缩小范围，精确到几个大区，甚至直接确定大区。此外，它们也可以帮你解决犹豫不决的两个大区和城市地区。

以 -P 结尾的黑码

有时，电线杆上会有一个黑色代码，包含数字和一个以字母 P 结尾的数字。通常可以在 Claro 电线杆上看到这些代码。虽然这方面的研究还不够深入，但我认为我们可以根据代码确定我们所在的区域：

2 位数字代码 + AMOV 文字（16P、17P）- 圣马丁

100 附近的 3 位数字代码 (152P) - 库斯科

200 左右的 3 位数字代码 + AMOV 书写 (231P) - 亚马孙

200 附近的 3 位数字代码 (273P) - 兰巴耶克

400-600 左右的 3 位数字代码 + AMOV 写法（470P、524P、693P）- 瓦卡维利卡，帕斯科

20000 附近的 5 位数字代码（26436P、27107P） - 皮乌拉

30000 以上的 5 位数字代码（33227P、74107P） - 利马

# 政党图案

如果你读过《酒吧长谈》等秘鲁政治文学，就会知道秘鲁的政治有多么混乱，秘鲁有多个地区性政党，各自具有独立的党徽。如果你能够知道这些大区独有的政党，看到政党就能认出地区的话，将会是非常有杀伤力的。

我尤其建议移动玩家和在锦标赛等比赛中需要移动作战的职业选手学习，因为大多数乡村地区的点位可能没有这些党徽。

# 景观

秘鲁在地理上可以分为三部分 最西边的沿海地区 中间的山地地区 和东部的雨林地区 在下文中 我将一一讲解这三块区域的辨别和细致区猜。

## 沿海地区

### Ruta 1(1号公路)

Ruta 1 是贯穿秘鲁整个海岸线的主要公路，从智利边境一直延伸到厄瓜多尔边境。南部路段标记为 1S（利马以南），北部路段标记为 1N（利马以北）。

猜测这条路所在秘鲁南北最重要的提示是，它从伊卡市分成两部分，并一直延伸到奇克拉约附近。

所以，如果你行驶在一条没有分隔带的沙漠沿海公路上，那么你就在该国南部（塔克纳、莫克瓜、阿雷基帕大区）。如果是在一条分隔带高速公路上看到类似的沙漠景观，则表明你位于更北部的地区（主要是安卡什和利马地区的部分地区）。

1 号公路沿线的农业区域（种植甘蔗等农作物）在北部（从奇克拉约到钦博特）最为常见，但在伊卡地区和利马地区的部分地区也有很多。

28B公路是亚马孙地区最具有学习价值的道路之一。全线沿河谷而行，且没有车道线。部分路段没有铺装。但大部分路段已铺装，无任何道路标线。
