# 第一部分：辨别不丹

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859817906-206c6ea7-8f29-4bcf-813f-d0751ef80676.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_29%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859818509-b6f1d17a-eace-4eed-b16f-b78fa32fee74.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹车牌几乎全都是红色的，这在街景中很容易辨别。

在亚洲，不丹是唯一全部使用红色车牌的国家。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692503246754-56cc4dcc-9946-4c67-b193-d65db713b84c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_18%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹的街景都是由银灰色无天线街景车拍摄。

所有使用灰色街景车的国家中，仅不丹为左行。需注意区分灰车和白车。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859819092-5f8f1151-15c2-4ea6-99c9-05e59138646a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹通常使用这些石块作为蜿蜒乡村道路上的安全屏障。这些石块非常常见，通常是识别不丹农村的最佳方式之一。

另一方面，常规的护栏很少见。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859820596-95738883-4ba6-4d88-b0a0-0aa05e67f92b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹传统建筑很有特色。大多数都是铁皮屋顶，刷油漆的木质框架，很有特点。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689418945885-1ec05c02-ab87-412a-a350-f2ffab1a2dab.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹语（或称宗喀语）是不丹的官方语言，其文字很有特点。

在街景中，这些字母只能在不丹找到。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859821777-d3b24bf1-266a-4631-a0b8-559dd5dfe3e6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹是个多山的国家，大多数街景都是在山区主路上拍摄的。

但该国南部非常平坦，见不到任何山地。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859822621-29c0620a-50f3-4ecb-890c-cc4391270f93.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹的道路旁随处可见经幡。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859823372-3ff127fd-fe5f-4e07-806b-2ddbff5ec4a1.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

偶尔会见到细金属电线杆，底部通常漆成黑色。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859823943-f69349dc-16b4-4f75-9fb4-85e2d6d35b99.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这些是不丹的路标，请注意黄色的顶部和白色的底部。

如果是在农村道路上，这些标记是帮助精确定位的最好线索。

在国道上，道路编号位于路标顶部，但地图上没有标示。不丹道路编号表格参见原文链接。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/52538357/1738941057529-e6ce84a8-ccf4-45b1-a824-5bc7bbaeae20.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹的城市中经常出现这种标牌，通常是红色或蓝色的。

如上图所示，标牌上通常会写着此处的地名。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859824917-81932c95-fdca-4033-9793-cdd2946f8750.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹的出租车有着黄色车牌和黄色车顶。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859825498-7f292675-e0c1-4bb8-bc6f-25b48d91c683.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

路杆通常有黑白相间的条纹。

在亚洲，斯里兰卡、马来西亚、孟加拉国也有这种路标。在猜测之前，请务必检查其他线索。

# 第二部分：辨别具体位置

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859826313-798e1177-0042-4ded-9272-1963f6facc4e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_19%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

不丹的大部分街景都位于该国主要道路和主要城市中。

建议牢记该覆盖范围。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859826977-17c7e772-b9a8-42a6-9484-3b6aa5ccf9ed.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

白色实线边线只出现于不丹西南部和靠近首都廷布的路段。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859827565-8f9b2d3c-53c7-4f05-8a58-e50d8f823b01.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在不丹，像这样的平坦景观只能在南部找到。

# 第三部分：特定区域技巧

## 机场覆盖

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859828078-1f011b71-0e71-4afa-802f-65b3f73043df.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

格勒铺机场（Gelephu Airport）是不丹唯一一座位于南部平坦地区的机场，很容易辨别。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859828608-5b645da9-bc1e-4435-9e56-3276073aa4b9.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

帕罗国际机场（Paro International Airport）位于西北部干燥的山谷中，是不丹北部两座机场之一。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678859829208-6b391a8a-6b90-49f1-91c8-aef8f581fcfc.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

而邦唐国内机场（Bumthang Domestic Airport）则位于北部湿润的山谷中。

译者注：不丹尚有另外一座无街景覆盖的永福拉国内机场（Yonphula Domestic Airport）。
