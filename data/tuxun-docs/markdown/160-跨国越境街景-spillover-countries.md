# 跨国越境街景

来源：https://www.yuque.com/chaofun/tuxun/spillover-countries

# 第 1 步 - 有关越境街景的信息

本指南包含所有有越境街景覆盖范围的国家/地区。

当一辆来自官方街景覆盖国家/地区的谷歌街景车驶过边境进入没有官方街景覆盖的邻国时，就会发生街景越境，从而导致该国家/地区获得一点官方街景覆盖。

另一种类型的泄漏是相机代数越境。越境覆盖的国家有官方街景，但没有特定的相机代数。

这对于国家/地区连胜最有用，因为越境国家通常被计算为越境国家，而不是街景车原属国家。

请您使用比谷歌地图更准确的OpenStreetMap进行边界核查。

但请注意：中华人民共和国的国界请务必参考中华人民共和国测绘法及有关地图管理条例。

图寻的匹配题库里不含这些越境的地点。

# 第 2 步 – 常规型越境

## 坦桑尼亚

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405501222-28a95069-57da-456a-8a4e-21bf77f0038e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

来自肯尼亚的坦桑尼亚Rongai边境越境可以通过土路变成铺砌的道路来识别，有黑白路缘石。沿着土路也可以看到许多松树。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405502060-15b81dd1-a064-46f9-ad76-7a93b05d1508.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在Namanga过境点越境处可以看到一个白色的小屋，后面有汽车在等着它。

## 冈比亚

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405502811-d90d3140-538b-420b-b1bc-3f54f16f61ac.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

塞内加尔Dimbaya越境到冈比亚有3条不同的道路。

- 一条东西向的铺装道路，看不到其他汽车或居民

- 一条乡村土路，沿路有栅栏，街景车拥有巨大的打码

- 一条狭窄但人口稠密的土路，有房屋。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405503391-06324e0f-3605-4ccc-96b0-b56b75945c78.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

N5越境到冈比亚可以通过红色的泥路肩，长得很高的草和街景车明显在很靠路边地行驶来识别。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405503988-fff429e5-717a-44e1-8690-bdcab1b644a4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在N4越境处可以看到一个浅蓝色的棚屋。

## 白俄罗斯

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405504680-69998d37-7b85-43cd-a15c-62a3ee5759bb.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

白俄罗斯维切布斯克（Viciebsk）和斯摩棱斯克（Smolensk）之间的P21越境可以通过黄色中心线，略带红色的土壤和傍晚街景覆盖来识别。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405505317-6a92497e-cb32-4e41-959b-beda74ebcdfe.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

斯摩棱斯克（Smolensk）以西的E30泄漏可以通过单黄线，正在建设的新铺道路和带有“ВИЦЕБСКАЯ ВОБЛ”的标牌来识别。（Vitsebskaya vobl.）在上面。

## 列支敦士登

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405506098-8950be29-3ce9-42a0-a3f9-373722f533d9.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在Buchs以东的桥梁越境处可以看到指向瑞士各地的顶部路牌和黄色虚线外线。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405506642-74116275-57fa-4d9a-b8c4-1177cce63d3e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在列支敦士登北部越境地点中可以看到这座带有大木屋顶的房屋和两个限速标志。

## 圭亚那

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405507403-33d60059-3cd8-4934-a0c9-f2bc7a348f6f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

里瑟（Lethem）以西的塔库图河大桥可以通过周围没有汽车和东边的驾驶侧交换区来识别。

## 巴拉圭

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405508062-dbf578a7-f7ea-44b4-8a21-145198fd2165.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

从巴西的Ponta Porã到巴拉圭的佩德罗胡安卡瓦列罗（Pedro Juan Caballero）有一条很长的街景越境道路。这条路看起来就像巴西任何城镇的很普通的道路，但偶尔可以看到巴拉圭国旗。

## 委内瑞拉

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405508718-c6a8cd8a-8192-498c-9652-1d04cc80b779.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

哥伦比亚库库塔（Cúcuta）和委内瑞拉Ureña之间的普恩特国际弗朗西斯科·德·保拉·桑坦德大桥可以通过黄红两色护栏以及带有红色底座的路灯来识别。这座桥总体上很繁忙。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405509413-d0bb34b7-80c4-4978-9757-d3c250a7d9b4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这条土路沙质干旱，前方有丁字路口，在Maicao以南可见。

![](https://cdn.nlark.com/yuque/0/2023/png/35193536/1689405510249-0b013558-57bc-4c5a-b272-846a6fbdafa0.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

BR-174从罗赖马州越境到委内瑞拉可以通过两国国旗以及写在上面的国家名称来识别。
