# 1 区域景观及特殊植被

## 1.1 Dehesa生态系统

译注：Dehesa是指西班牙的一种农林业耕作生态系统。它们是多用途的开放林地，主要通过砍伐天然林而形成，结合了牲畜饲养、谷物种植、木柴收获以及狩猎等功能。

Dehesa景观以成片稀树（圣栎和栓皮栎）为特色。

![pic-2.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712981843761-38e05c6a-62ad-4308-aa8d-71b962a01a5e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

植物注：

![pic-1.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712981050378-794ae7db-81be-4527-a60b-73190212eca2.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

- 圣栎/冬青栎（Quercus ilex）：常绿乔木，树冠球形，枝条高度较低，树冠占植株比例很大，叶卵形，有刺。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38693261/1748420721649-a01ad42a-3810-4638-8130-50ff10304a26.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_50%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

补充：应把该种同橄榄区分开来，区分的关键要素是枝条和树干，橄榄的树干有明显纹理，(幼体树皮光滑)枝条弯曲，叶颜色也较浅，而圣栎的枝条平直，树皮光滑。(下附橄榄图片)

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38693261/1748420803930-af977404-9e17-45ff-833c-96e9c61e13bf.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_46%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

- 栓皮栎（Quercus suber）：常绿乔木，枝条弯曲，树干弯曲，树皮深裂，红棕色。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38693261/1755237842354-4969cb1e-0802-464e-827a-c55528f7fdb4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_42%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

岩壁在卡塞雷斯省最常见，也多见于萨拉曼卡省和巴达霍斯省；不太常见于韦尔瓦省，不会出现在塞维利亚省和雷阿尔城省的东南部。

题外话：萨拉曼卡省的三代覆盖几乎都是冬景。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712811244032-284b2275-071f-4add-8ba4-1908f45d164d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_15%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

## 1.2 葡萄种植区

西班牙很多地方都种植葡萄，但如下所列是最重要且最有用的。

### 1.2.1 拉曼恰高原

拉曼恰高原位于马德里以南，红土与葡萄分布广泛。这里通常非常平坦、景观有时比较干燥。三代街景大多覆盖于2013年3-4月，而四代也是在春季拍摄的。

![未标题-2.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712812722724-8756b06b-ecb0-4414-b0b2-ffe2ce64cab4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_53%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

此外，您还可以在高原东南部的阿尔巴塞特省一带找到风车。这里除了葡萄园外，果树和意大利五针松也较多。

### 1.2.2 佩内德斯

Penedès

![](https://cdn.nlark.com/yuque/0/2025/png/38693261/1755238754578-b1492abc-e14a-4e43-8d11-b6afe4b9c752.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_19%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

## 1.4 欧洲赤松

Pinus sylvestris (pi roig)

译注：就是我们常说的“波罗的海松”

该树种树干为红色，非常独特。

![cee5646151347adc3a0794d731affb69.png](https://cdn.nlark.com/yuque/0/2025/png/35359367/1735961435746-f9e5689a-c0bd-4c03-aa61-fc9bf401b8ef.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_28%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

主要有四大分布区域：

- 比利牛斯山脉：崇山峻岭之中，森林更茂密，植被也更多。

# 2 覆盖及天线

西班牙有着大量的街景覆盖，街景车种类繁多。不过在一些地区，它们的大部分覆盖拍摄于一小段时间。下面我们来介绍一些最独特、最有辨识度的覆盖。

## 2.1 短天线（三代街景）

## 2.2 长天线（三代街景）

长天线在西班牙随处可见，2012年在全国范围内拍摄，不过数量不多，主要在主要道路和城市。这些覆盖大部分都被替代了。在阿斯图里亚斯、加利西亚、卡斯蒂利亚－莱昂和安达卢西亚，它们出现概率的要大一些。

在穆尔西亚、加纳利群岛、巴利阿里群岛、埃斯特雷马杜拉和拉曼恰，没有长天线的分布。

巴斯克地区和纳瓦拉有一点长天线分布，同时也有中天线。

![_____ (1).png](https://cdn.nlark.com/yuque/0/2025/png/38693261/1736001345611-fc73403e-4ca1-450c-b336-2026b4998bd9.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_183%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

### 2.2.1 阿拉贡

阿拉贡是唯一有着大量长天线覆盖的地区，而且这些覆盖还没有被替代。阿拉贡大部分地区都很干旱，因此如果看到这种野外环境+长天线覆盖，几乎能判定在阿拉贡。

## 2.3 中天线（三代街景）

中天线如其名，长度介于长天线、短天线之间。和长天线一样，中天线也能看见街景车尾部的黑色。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38693261/1736000601925-72345732-2bcd-4962-b353-11a47751159a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

该天线比较罕见，只出现于北方，如纳瓦拉、巴斯克地区、坎塔布里亚、阿斯图里亚斯、 加利西亚 ，以及特鲁埃尔的小部分地区（因为比较罕见，作者没有过多研究，这些信息可能不准确）。其中主要是在纳瓦拉。

## 3.4 Km公里标

Sergiotor9 制作了一份对学习西班牙公里标极具指导意义的文档，原文链接如下：[Comprehensive list of Spainish km markers](https://docs.google.com/document/d/1Gm9yf5_530K5kGN00xe_GEW_43akMfKftBypJ8XXi2Y/edit?tab=t.0#heading=h.c91f1i74w4l3)

译者注：由于篇幅原因，此处仅对文档中有关总体概括以及使用方法的部分进行翻译，其余内容请读者阅读原文进行学习~

​

西班牙公里标的通用设计是彩色的矩形或正方形，颜色与道路级别相对应。

- 国道始终使用红色标记牌。

- 高速公路始终使用蓝色标记牌，即使它们是某些城市或地区独有的。如果高速公路属于欧洲公路网络的一部分，它们则会因为顶部的绿色标记而呈现矩形，否则，它们大多呈正方形。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38693261/1755171727761-5ec65380-8059-4bb6-a8e6-d04158493b8a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_50%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

西班牙的自治区（下文简称“AC”）在其道路网络方面存在一些显著差异，例如，巴斯克自治区没有自治区级道路，而阿斯图里亚斯的每个市镇都有各自不同的编号。

一般来说，公路会遵循如下的颜色标识方案。在固定轮次（No Move）中，这对你寻找一条名字不认识的道路很有帮助。

- 一级公路使用橙色标记牌。这些道路的代码代表整个自治区，并且通常有完整的道路线。如下图所示：

![屏幕截图 2025-08-11 222522.png](https://cdn.nlark.com/yuque/0/2025/png/46298828/1754922389886-a48de72f-dab0-4d43-b0ed-bab92953cb82.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_49%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

- 二级公路使用绿色标记牌。这一级别最为少见，因为有些自治区完全没有这类标牌。其代码代表所属自治区或者省份。多数二级公路有着完整的道路标线，但也可能会出现没有中线或没有边线的情况。如下图所示：

![屏幕截图 2025-08-11 235745.png](https://cdn.nlark.com/yuque/0/2025/png/46298828/1754927907658-e3e6d7e3-fa0e-4702-ba06-c88d6fa3371e.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_54%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

- 三级公路使用黄色标记牌。这些道路的范围较广，可以是省级道路，也可能是非常小的市镇道路。单行车道很可能也属于这一级别。如下图所示：

### 3.4.1 如何在游戏中使用公里标

本指南中信息最重要的用途是，让你在固定模式中，即使遇到一个无法清晰辨认的公里标时，也能判断出其可能所在的自治区。此方法主要适用于连胜模式（streaking）（如果你还没试过，强烈建议亲自体验一下），在常规题库和匹配模式中当然也同样适用。

例如，在一轮对局中，出现了一片dehesa景观，那么埃斯特雷马杜拉、卡斯蒂利亚-拉曼恰、安达卢西亚或卡斯蒂利亚-莱昂都是可能的答案，但如果还有公里标，我们就能仅凭其设计判断出街景所在地，因为这些省份的设计并不完全相同。（严格意义上，雷阿尔城省的市镇级道路使用与卡斯蒂利亚-莱昂相同的黄色矩形标志🤓，但除此之外，其他任何公里标都能在无需读取编号的情况下告诉你正确的自治区。）

### 总结

路肩颜色meta非常有助于区域猜测。但我无法保证100%正确率。
