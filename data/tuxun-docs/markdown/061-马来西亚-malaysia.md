马来的车牌是黑色的，经过打码处理后一般也能看到两块白。

印尼的车牌也是黑色的，但是它们会有三块白。

![Road+lines.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641143505-e5c6df84-fcf6-433f-a642-daf4ee22ff69.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

马来在其双车道公路上经常使用双白中线，这是东南亚唯一这样做的国家。

![yellow+road+markings.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641146766-6ca18e17-9793-4891-817f-2037d88e53be.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这种粗黄带是马来独有的。

虽然不那么常见，但在中国香港和英国也可以看到这种样式。

![新建 Microsoft PowerPoint 演示文稿.jpg](https://cdn.nlark.com/yuque/0/2023/jpeg/35193536/1699707040208-ae8d9eb3-7806-4efc-9aa5-f6a23252869b.jpeg?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_36%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![Untitled+design+(25).png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641151659-016654f1-570f-4c4e-bbd6-3aca1ab40b73.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

马来的电线杆上经常会贴有黑色或白色的贴纸。

黑色贴纸下面有时还会有红白或红蓝相间的条纹。

![Untitled+design+(24).png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641170860-b0e16f0e-cb2c-453a-ac12-41b36fa032be.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

BERHENTI 停车标志和 BERI LALUAN 让行标志都是马来独有的。

![Untitled+design+(23).png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641176265-ac115903-2801-4f04-a207-decb8a063e36.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

马来有各种各样可能包含黑、白、灰、红的路桩。以上是三种常见的马来路桩样式。

路桩样式不同，但布局大多相同。

![Bild_2022-06-30_215703530.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641183308-0d76aa49-1a40-4982-9191-902d6f726881.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这是马来最为常见的指路牌样式。黄色六边形高速公路标志是马来独有的。

![Untitled+design+(30).png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641249142-a2c20edd-0365-436c-a598-c41b97dbfcf8.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

马来的道路公里标有其独特的白蓝配色，其上经常出现六边形公路标志。

![my_language.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688640447887-9c8e4f34-0c54-456c-90c9-ab98e27c701b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

马来语使用拉丁字母书写，它与印尼语密切相关。不说这两种语言的人实际上无法将其区分。

![Bild_2022-06-26_205913750.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641254356-a187feca-3ad6-4566-900b-0613044e41e4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

与邻国印尼相比，马来出现中文的机率大很多。事实上，马来人有24.6%是华人。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1699708328395-8574a115-f867-4102-9eb5-b4b7a7fd6cfd.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

马来的路标杆几乎永远有黑白相间的条纹。

在亚洲，这种路杆样式也见于斯里兰卡、不丹和孟加拉国。

![Copy+of+AUshuaia}(6).png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641264266-0888297e-ed0f-431a-b7e9-4499e40a7f4c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

黑白相间的路缘石在马来很常见。

许多亚洲国家也会有这种黑白相间的路缘石。

# 第二部分 - 地区和州特定线索

地名对照表

- 马来西亚半岛（西马/ Peninsular Malaysia/ Semenanjung Malaysia）

- 沙巴（Sabah）

- 砂拉越（Sarawak）

- 古晋（Kuching）

- 柔佛（Johor）

州旗有时能看到，所以它们值得学习。

马来西亚的邮政编码有必要了解一下，因为在当地的大多数路牌上都能看到它们。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1699709317441-b509c02d-ee98-4fc0-a4e4-8b34a9c2d273.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

灯柱带有三角形金属支架的主要分布在马来西亚半岛。

![Diagonal_Support_Poles.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641301729-523d849c-8c85-46e5-a954-1b0bf2935681.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

婆罗洲一侧的电线杆通常具有2-1的绝缘子配置和长对角木支撑。它们只会在砂拉越出现。

印尼的电线杆可能看起来相似，但印尼的电线杆通常不是木制，并且印尼的电线杆的对角支撑更小（甚至有可能没有）。

![Two_Horizontal_Support_Poles.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641306638-d6d45813-6b7b-4806-993d-a13d40897391.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

顶部有两根平行杆的细金属电线杆只出现在沙巴。

![Sarawak+Lamp+Posts.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1688641310575-81f746e1-ff8a-4660-9219-1bc0ce672d43.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

这种有一个形状像回力镖的金属在支撑着的灯柱可以在砂拉越找到。

它在古晋以外的城市中更常见。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/52538357/1739102513957-e05ac8da-1824-486f-a5e2-fb7429a56a59.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

有着类似字母“S”的标志的垃圾桶可在森美兰州、马六甲州和柔佛州发现。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/52538357/1739102641241-9ec5ce55-f8f3-4cf2-a84b-ed3af0832276.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_43%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

有着类似叶子标志的垃圾桶可在玻璃市和吉打找到。
