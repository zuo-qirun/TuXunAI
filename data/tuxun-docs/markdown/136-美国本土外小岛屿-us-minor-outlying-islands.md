# 第1节 - 鉴别岛屿

![midway_countrystreak.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300010478-7b1c1326-b1ba-4938-a0c9-ff6135b6240f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_57%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

美国本土外小岛屿是一串美国岛屿领土的合称。在国家连胜模式中，它们被视为美国。

本指南会有些偏向中途岛，因为那是唯一有可定位覆盖范围的岛屿。

### 何谓“美国本土外小岛屿”？

![United_States_Minor_Outlying_Islands.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697303400381-6029b1c3-302f-48ad-9b6b-85a679fd49fe.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_29%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

美国本土外小岛屿（United States Minor Outlying Islands）是由ISO 3166-1所定义的、美国所属的九处未通过组织法的岛屿和环礁之统称，它们包括：

- 大洋洲 - 波利尼西亚岛群

- 中途岛（Midway Atoll），位于夏威夷群岛的西北方。

- 唯一拥有谷歌街景官方覆盖的岛礁，这也是为何原指南省略了余下的岛礁。

- 实际上是一群珊瑚环礁，包括主岛沙岛（Sand Island）与较小的东岛（Eastern Island）。

- 约翰斯顿环礁（Johnston Atoll），位于檀香山的西南方。

- 豪兰岛（Howland Island），位于凤凰群岛（Phoenix Islands）北部。

- 贝克岛（Baker Island），位于凤凰群岛北部。

- 金曼礁（Kingman Reef），位于莱恩群岛（Line Islands）北部。

- 巴尔米拉环礁（Palmyra Atoll），位于莱恩群岛北部。

- 唯一一处未通过组织法的合并领土。

- 贾维斯岛（Jarvis Island），位于莱恩群岛中部。

- 唯一位于南太平洋的岛礁。

- 大洋洲 - 密克罗尼西亚岛群

- 威克岛（Wake Island），位于马绍尔群岛的北方。

- 北美洲 - 加勒比地区

- 纳弗沙岛（Navassa Island），位于大安的列斯群岛中部。

### 那么原指南的附图实际上是？

![midway_countrystreak.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697301656700-f7bf988d-170e-48e8-bebf-88397ad37fc6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_57%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

西北夏威夷群岛。将它视为美国本土外小岛屿是错误的：除去中途岛，剩余的岛礁均属夏威夷州管辖，而非美国本土外小岛屿。

![midway_gen2.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300685492-22a2872d-7d85-436a-9339-e5ddca2da7a4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

中途岛是二代相机覆盖。

![midway_albatrosses.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300683623-196ab178-116f-4b91-83a8-ee4ed6aa5fb8.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

中途岛最显著的特点之一是：几乎每个位置都有多只信天翁。

![midway_runways.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300687019-6c2191e4-8b7b-46d7-8232-67845f7b464f.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

中途岛的大部分覆盖是在两个跑道上拍摄的，这是唯一可能难以发现信天翁的区域。

![midway_otherislands.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300683682-ee52ccbc-a31f-4d27-92c1-2d2fd9d494cf.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

除了中途岛以外，其他岛屿常具有下列特征：

- 三代徒步覆盖；

- 低矮灌木植被；

- 大量鸟类存在。

# 第2节 - 区域猜测

![midway_nicerunway.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300686876-73e6ed0b-6529-49ab-b203-e227a8691320.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

中途岛南侧有一条铺装良好、从西到东延伸的跑道。

![midway_roughrunway.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300687164-9639965a-382d-4c8e-87c9-bd6a35a7713d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

另一条铺装较差的跑道从北到南延伸，越偏北越差。

![midway_town.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300685576-98972b5e-9ed9-425c-93af-dd835e466568.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

中途岛东侧有一个小村庄。

![midway_southeastisland.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300683330-bd74786c-2bdf-40d3-b1d3-4ee25c3dd62a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

东南岛可称得上是这一串岛屿中最不显著的岛屿。它的中心附近有一个小湖，天空中有大量飞鸟。

![midway_lisianskiisland.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300685621-37cb440f-b581-4952-929c-bd874e9445c3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

利相斯基岛只有岸边覆盖，整个岛屿周围分布大灌木丛。

![midway_laysanisland.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300685637-d67e55dc-5b99-4970-b575-b389423dc283.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

莱桑岛是这一串岛屿中唯一万里无云的岛屿。

![midway_frenchairport.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300683320-2de02f37-5370-4acf-be54-e458da98ac77.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

法国战舰浅滩是一座相当狭长的岛礁，有一条看起来破旧的跑道。该岛屿是这一串岛屿中海鸟最多的。

![midway_eastisland.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697300685660-7bdd5247-59e1-476e-97e8-0f221ddd9056.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

东岛 East Island 是这一串岛屿中最小的岛屿。几乎没有植被，可以从各个方向看到海洋。
