# 圣多美和普林西比

### 第一步 – 识别国家

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712795184974-5f47e7b5-763e-467e-a6ba-d89e1931c582.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712795188086-addcccac-914f-483d-b2a1-ddcc10a86cf3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

圣多美和普林西比所有的街景都是用画质低的“印度相机”拍摄的。街景车被完全打码，前端的打码面积稍小而后端打码面积很大。

该地区的街景覆盖使用的是印度相机的一个新变种。用这种印度相机拍摄的街景相较于印度和尼日利亚早期用印度相机拍摄的街景更暗、更模糊。目前使用该种相机的只有该地区以及尼日利亚。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712795186012-c1885a62-9cf6-4fbe-8587-5c9efb41a0a3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

圣多美和普林西比主要使用葡萄牙语。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712795188976-3b41a035-0f14-47b1-8aed-fc4b89bb1e44.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

除了语言之外，在该地区基础设施中也能看到葡萄牙的影子，比如与葡萄牙设计相同的带孔电线杆以及斑马线标志。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712795191079-4f43d3b0-fbd7-48bc-9967-7252ee95c75a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

圣多美和普林西比是热带岛屿，几乎完全被雨林覆盖。 香蕉在整个岛屿上尤其常见。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768923930584-491cdb23-d2bd-4b01-b874-4dde049d369c.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在很多地点，车的侧面会伸出一根斜着的天线。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712795189196-901013aa-6673-4ce1-be78-7eaa3f0cd050.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

一些木头电线杆底部涂蓝，电线杆上能看到“CST”三个字母。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768924005066-03b93e67-f77f-4aeb-8c92-f68bbd4ed27f.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

出租车和迷你公交的涂装是亮黄色的。

### 第二步 – 区域特点

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712795191247-75883710-7efa-4733-83a5-44a2fa05bfb4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

用“印度相机”变种拍摄的更暗、颜色饱和度更低的街景在普林西比岛上最常见。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768924068436-d8f42457-0adf-4455-b92a-1696a0cc41d3.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

普林西比岛的街景车的车顶是蓝色的，带有银色的行李架纵梁。

注意它有时候是全码的。

![](https://cdn.nlark.com/yuque/0/2026/webp/52538357/1768924191095-734b7f5d-32b0-4ba6-9df1-b7ea82a2e2fe.webp?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_34%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

圣多美的街景车是黑的，行李架纵梁也是黑色的。大多数情况下，这个街景车被完全打码了。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35193536/1712795191867-0a88922c-2326-4ba4-94fe-df3b8da7d46c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

您可以在圣多美南部看到油棕种植园。
