# 第一节 鉴别皮特凯恩群岛

![2560px-Pitcairn_Islands_in_United_Kingdom.svg.png](https://cdn.nlark.com/yuque/0/2024/png/34598262/1705576120179-be11ed36-2912-4ae4-a689-c368d5c7ea41.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_73%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

皮特凯恩群岛由南太平洋上的四个岛屿（皮特凯恩、亨德森、迪西和奥埃诺岛）组成，是英国在太平洋唯一的海外属地。

![pn_coverage.png](https://cdn.nlark.com/yuque/0/2024/png/34598262/1705575977176-b5185e15-d8da-4b0f-8bb2-600769f92a15.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

覆盖仅存在于其中的两个岛屿： 皮特凯恩岛和亨德森岛。随后的指南将以皮特凯恩岛为重点，因为亨德森岛的海滩覆盖范围很小。

![pn_trekker.png](https://cdn.nlark.com/yuque/0/2024/png/34598262/1705575981894-73abf469-8ef9-4d1d-8112-e12bc7d37300.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

所有街景都是由三代徒步者拍摄的。

![pn_landscape.png](https://cdn.nlark.com/yuque/0/2024/png/34598262/1705575984292-a87c57dc-4423-4b66-b1a6-81f0d0761e86.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

皮特凯恩岛是一个树木茂密的半热带岛屿，大部分覆盖都是在穿过森林的小土路上拍摄的。

![pn_soil.png](https://cdn.nlark.com/yuque/0/2024/png/34598262/1705575986315-f1e7af04-afae-49ed-9e51-be00dcb19cf1.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

土壤通常为红色。

# 第二节 区域线索

![pn_adamstown.png](https://cdn.nlark.com/yuque/0/2024/png/34598262/1705575994642-64c0ee6f-5015-4a69-924c-89d99f554f90.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

亚当斯敦是皮特凯恩岛上唯一的定居点，位于东北海岸附近。穿过该镇的一些道路是用混凝土而非泥土铺成的。

![pn_henderson.png](https://cdn.nlark.com/yuque/0/2024/png/34598262/1705575988588-fe685cd6-8776-4002-926c-f91b7f736f2d.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

亨德森岛只在北岸和东岸的两处白色沙滩有覆盖。
