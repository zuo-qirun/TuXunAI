# 第二部分：辨别区域和行政区的线索

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162869319-9b76dd9c-0d7e-48fd-b69f-b4bf2327bcf6.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_58%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

孟加拉国有几种差别很大的屋顶风格，方便区分该国的东西部。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162064235-06f69155-67af-4384-8830-7dc3d154c64b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在西部，特别是Rajshahi专区、Khulna专区和Dhaka专区（达卡），以金属板制成的斜屋顶为主。Rangpur专区和Chittagong专区（吉大港）有时出现这种屋顶。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162064337-4f19036a-27f1-4db4-a231-ef54b1baee00.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在东部，金属板屋顶的屋脊下方通常有一个三角形的“窗户”。这种样式集中于Sylhet，在Chittagong、Mymensingh、Rangpur东部和Dhaka北部也可见到。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162066455-91750154-bfb8-48ac-b3d0-e489a4743d39.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

瓦片屋顶在孟加拉国并不常见，而且都是从印度的西孟加拉邦传入，因此只在Khulna的边境上发现。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162066359-2e8deb04-54b9-451c-aa26-886f4e2456c9.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

水稻是孟加拉国的主要农作物，每年种植两次。根据水稻的高度和颜色可以区分生长期，这对区域猜测非常有用。

几乎不伸出水面的绿色矮水稻属于幼苗期。分布于Khulna、Mymensingh和达卡的博多河（即恒河）以南。

注：请注意，此内容和以下提示仅适用于第4代相机。

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162066247-b0df71c4-7191-4c05-a69c-2b466e00b92a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在Rangpur、Barisal和 Chittagong可以看到稍高的水稻，处于生长中期，仍然是绿色。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162069599-99cfecbc-ee6c-473a-8bf3-35cf1ff17dea.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

成熟期水稻的特点是明显更高，颜色为黄色或橙色，尖端更蓬松——蓬松意味着水稻已经成熟。主要分布于 Rajshahi，但Rangpur和 Chittagong也有。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162069147-92e0a7de-b72d-4815-b094-3bcee89a7c82.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

已收获的水稻可以通过其米黄色或棕色，以及较短的、被收割的状态来识别。有时周围还会出现成熟期、未收割的水稻，甚至是新种植的幼苗。这在 Sylhet和 Khulna南部最为常见。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162070638-8ff2d65f-fca0-4ff6-b4a1-9210c2d9eade.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

稻田需要一些时间来休养生息，所以自然会遇到一些处于农闲时节的荒芜、空旷，完全没有水稻的田地。这种主要位于 Dhaka的博多河以北。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162070183-a5ead2ae-1e09-40cd-9def-79c0f0c6fc85.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在Dhaka、Mymensingh和 Sylhet的交界地带可以看到广阔的稻田，几乎没有其他任何东西。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162071568-f60464c8-daf5-4dc4-8665-ad2f9d2d28b8.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

虽然孟加拉国大部分地区近乎完全平坦，但在吉大港东部，你可能会发现相当高的山丘，上面森林茂密、未经开垦。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162073304-ab828673-9b8e-47f6-9d41-6910af7befef.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

从 Sylhet北部和 Mymensingh东部的印度边境可以看到一条相当庞大的东西向山脊。它经常被浓重的雾霾掩盖住全部或部分山体。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162075536-c853a02d-8053-480b-b140-18ce4e7b5ab5.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

玉米通常种植在孟加拉国西部，Rajshahi、 Rangpur，Mymensingh西部和 Khulna北部都有大中型玉米田。

在Barisal的南部可以找到较小的玉米地，通常与其他作物相邻（如下图）。

![d433253ef610331e106acd5f0c68a7e1.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729164531542-3747da5d-7d80-4273-b8f9-43b34a424485.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_17%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162075638-b5b3e3ab-039c-47a5-b069-cd8f9c494a9b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

黄麻是一种高大的绿色作物，干燥后可用来制作地毯、袋子和其他物品。由于街景拍摄的季节，黄麻几乎只出现于 Rangpur的街景中。干燥后的黄麻，它可以挂在围栏上，或是在全国各地的道路上晾晒。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162077029-80bae7ab-b4e7-4b2f-95f8-59040c57614b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

油菜是一种低矮的作物，开黄色花朵，用于制作食用油。它通常在水稻的种植间隙栽种，因此它在 Dhaka的帕博多河以北非常常见。

​

![image.png](https://cdn.nlark.com/yuque/0/2024/png/35359367/1729162076635-89ecb742-c260-4462-86db-e6574e64797b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

孟加拉国的茶叶种植集中于 Sylhet南部的高地，吉大港也有极少数分布。
