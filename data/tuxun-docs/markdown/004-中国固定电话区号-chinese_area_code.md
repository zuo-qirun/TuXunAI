# 固定电话网络

本文主要介绍中华人民共和国境内民用固定电话网络的有关内容。

固定电话号码的基本格式0AX-CXXX-XXXX （大区中心局(C1)城市使用）
0BXX-CXXX-XXXX（大多数非C1局省会城市、经济较发达城市使用）
0BXX-CXX-XXXX（其余城市使用）

- 结构上，0为长途冠码，并非长途区号的一部分。

- 有效固定电话号码为7位或8位数字；第一位数字为2-8，其它数字为0-9。

- 国外拨打中国固定电话时，需加国际电话区号冠码：

- 香港特区 +852

- 澳门特区 +853

- 中国大陆 +86

- 台湾地区 +886

### 位数

大多数城市的电话号码为7位，但以下城市的电话号码为8位：

- 华北大区：北京、天津、石家庄、郑州、开封、南阳、洛阳；

- 东北大区：沈阳、抚顺、铁岭、本溪、大连、长春、吉林、哈尔滨；

- 华东大区：上海、南京、江苏全境、济南、青岛、合肥、浙江部分↓、福州、泉州、台湾部分↓；

- 浙江：杭州、嘉兴、宁波、绍兴、台州、温州、金华；

- 台湾：台北、新北、基隆、台中；

- 中南大区：武汉、鄂州市华容区、广州、长沙、株洲、湘潭、广东部分↓、南昌；

- 广东：汕头、深圳、佛山、东莞、中山；

- 西南大区：重庆、成都、眉山、资阳、贵阳、遵义、安顺、昆明、海南全境；

- 西北大区：西安、咸阳；

下划线代表该地使用3位区号。
