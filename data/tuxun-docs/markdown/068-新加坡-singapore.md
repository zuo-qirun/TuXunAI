# 第一部分：辨别新加坡

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678276456815-1682cf17-c80a-4077-be51-4ca905544c51.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_29%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678276457434-08eef7d2-3c4e-4378-9045-0ee1e16e1075.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_21%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

新加坡是一个高度发达的城市国家，其特点为城市整体整洁且有序。

主路通常有分隔带，通常有黑白相间的路肩和双黄色边线。附近还可见到高耸的公寓楼。

大多数情况下，这种“整体气质”即可帮助确定新加坡。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411554396-d54fc269-8edb-4b4f-b313-b78088336744.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_39%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

新加坡所有路名牌皆为此特殊样式，绿色底色，边角圆润，白色字体。

路名通常源于其所在的小区域名称。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411518447-f2788c5a-2020-4717-8e59-b146f4753465.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

新加坡大多数车牌为黑底白字。也有部分英式的前白后黄车牌，以及非常罕见的红底。

附近的马来西亚和印度尼西亚同样使用黑底白字的车牌。不过，这两个国家车牌上的文字通常分为两节（马来）和三节（印尼）。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411520405-ee3a1123-c618-4a59-b52b-65f35bc4c79c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_28%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

指示方向的路牌同样为绿色，文字为英文。

高速公路都有唯一的三字编码（如图中的TPE），可轻易辨别。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411554775-6e4a7b98-4e2d-495c-83c3-6ed37df1be75.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

新加坡的路标很独特，因为它们都有白色的方形牌背。

除澳大利亚和新加坡外，基本没有国家使用这种方形牌背。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411642954-1d8ac1a9-73f1-4208-a121-912b51d1f1b8.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_40%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在新加坡各地都可以看到热带雨林树种，通常沿着主要道路种植。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411675928-eca3c9f8-cc53-41d3-8a3a-e467af4acf5b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_46%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

警告通常用大号字体写在路面上，如著名的“HUMP AHEAD”（前方路不平整）。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411705726-35bf9e59-e941-4f37-a633-87bf5849ebf2.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

新加坡的公交车大多是青绿色，因此很容易辨认。也有其他涂装，例如白紫色的SBS Transit变体。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411722378-dbb7a9fd-75a6-421e-9c2e-4e65538b33e2.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_45%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

此外，所有公交车站都有混凝土路桩保护，护柱上有独特的黑黄色警示带。

马来西亚一些城市也有类似设计，但并不完全相同。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411757251-3e5b48b1-25c2-4442-90f6-49c6ecb6d05b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_71%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

除了混凝土路桩，新加坡部分公交站另有路面上的一块黄色公交启动区域，里面有个箭头。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411818781-18f7bd44-2795-432b-afaf-b157747102dd.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_52%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

几乎所有新加坡交通灯都是这种由带几个洞的弯曲金属支架支撑。

# 第二部分：辨别具体区域的线索

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411947831-0e5ddb32-8d65-47f1-ad44-464846c195c4.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

许多街道都以其所在的街区命名，尤其是郊区。

这些街区的名字在地图上很轻易就能看到，因此可以在很短的时间内达成5000分。例如，图中的标志指的是宏茂桥附近的31号街道。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692411996448-c4f891ee-9072-4c10-9943-be10ef3c5c75.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

Tuas（大士）规划区是一个工业区，其特点是笔直的道路、大型工厂、起重机和航运集装箱，易于辨别。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692412035770-ff93c2be-1968-4c37-8870-d2ec56b24ad3.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

圣淘沙岛有着绿色路标柱、蜿蜒的道路，且给人一种度假胜地的感觉，可轻易辨别。部分路段还有这样的绿色车道。

![](https://cdn.nlark.com/yuque/0/2023/png/35076970/1678276465657-cad6c471-a390-4aa2-b021-deb6891a4747.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_18%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

# 第三部分：小技巧

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692412110094-87ef9f2a-cd3d-44cf-9678-3fb1d1e833ee.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

乌宾岛（Pulau Ubin）上的第三代徒步拍摄街景可通过特殊的乡村植被覆盖，以及拍摄者偶尔露出的迷彩帽识别。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692412144350-35843a2d-e8b7-425f-aaa0-aa661df9d96b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

可通过第三代徒步街景相机、平坦的柏油路和几乎总是在旁边的大海，来辨认圣约翰岛和拉撒路群岛（St John and Lazarus Islands）

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1692412177517-0b43129e-b65e-4720-9f5d-15ee134b9d41.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_63%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

小小的库苏岛很容易辨认，因为所有道路都用了红色铺路石。
