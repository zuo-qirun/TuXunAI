# 前言

不丹，凭借其在匹配题库中超高的配比、辽阔的疆域和高难度的区猜技巧，日益成为图寻玩家关注的焦点。（并不是）

然而，很明显的是，不管是在国内的图寻社区，还是在国外的geo社区，都没有一个较为完整的不丹区域猜测教程。于是笔者便怀着尝试的心态，对不丹的街景覆盖进行了一定的整理与归纳，写成此篇文章。

在本篇文档内，技巧优先级大致为 街景车技巧 (Carmeta) > 文字信息 > 环境，因此建议以“先看车，再找文字，最后看环境”的顺序进行区域判断，下文文档的主体部分也将以此顺序展开，并以各技巧的重要程度进行排序。当然，就不丹进阶学习者而言，环境显然是优于其他项的。但是……真的有人愿意深入学习不丹吗？

文档内所有技巧的分布范围皆采用图寻所使用的华为地图展示，严格遵循国家相关法律法规，符合国家地图规范。

# 第一部分 —— 街景车

不丹的街景车车身为银色，一般仅露出尾部部分。可以通过车尾打码的程度，分为以下三种打码情况。

![3.PNG](https://cdn.nlark.com/yuque/0/2025/png/38682438/1752902913588-a2ea3d96-9d30-4cf8-a7ae-5bee2caff888.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_40%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

普通打码是最常见的打码形式，一般仅露出车尾部分区域和红色尾灯。

![1.PNG](https://cdn.nlark.com/yuque/0/2025/png/38682438/1752903073443-e4666fc9-1de6-4109-a785-ea8029e895eb.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_41%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

高位打码仅露出车尾和尾灯的边缘，有时甚至完全被打码，无法为我们提供任何信息。其常见程度仅次于普通打码，也有很广泛的分布。如果遇到高位打码的街景车，请直接跳过观察街景车的步骤，并直接进入第二部分——文字信息。

![2.PNG](https://cdn.nlark.com/yuque/0/2025/png/38682438/1752903715314-706736bb-8f71-451c-84e9-7883f8e6351c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_38%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

在低位打码中，街景车的后部全部可见，甚至能够露出部分车顶，较为少见。

在三种打码中，仅有在普通打码和低位打码时，才可能实现区域猜测。而在不丹，街景车的打码方式总是飘忽不定，或许在几步间，低位打码就变为高位打码，可谓“失之毫厘，谬以千里”。

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38682438/1752904630687-664d3cb0-aa1f-43d5-8c9a-ad68d67f0002.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_40%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

![image.png](https://cdn.nlark.com/yuque/0/2025/png/38682438/1752904664884-1850857e-32d8-4b3a-9740-fd74838ab512.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_38%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

如果无法从街景车上找到任何线索，或是遇到反光太强、实在看不清等不得已的情况，请直接跳过，并寻找文字信息。部分线索较为细微，因此不丹的街景车技巧需要较好的视力支撑。
