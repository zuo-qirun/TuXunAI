# 第1节 - 鉴别关岛

![gu_location.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697290575732-c0dc61a2-4aca-4629-a0f7-6c4862b709f0.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_68%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

关岛是美国的海外领土。因此，在国家连胜模式中，它被视为美国。

![gu_car.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697290584167-c2037504-8914-4dc2-9ebc-4faeea54cb4b.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

整个谷歌街景车都处于打码状态，但是仍然可见右前方的天线和一部分车顶行李架。

北马里亚纳群岛的街景车上看不到车顶行李架。

![gu_pole.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697290594479-0951bdf0-cfc8-4232-8067-8a69db438355.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

关岛的电线杆较粗，且总是混凝土材质。

北马里亚纳群岛一般为木质电线杆。

![image.png](https://cdn.nlark.com/yuque/0/2023/png/35193536/1698629934401-ccf50607-dcc6-4342-8c98-8aacf0fc35e5.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_19%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

关岛使用绿色路牌，而北马里亚纳群岛则使用蓝色路牌。

![gu_landscape.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697290636502-6cadf0a4-5b45-4c7a-b95d-982ce5ad1b1a.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

关岛属于热带气候区，有很多茂密而相对低矮的植被。

![gu_reflectorspam.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697290644754-7eaa969e-f1f9-4ce3-98ef-28f496c0994c.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

关岛和北马里亚纳群岛常有很多道路反光镜。

![gu_asian.png](https://cdn.nlark.com/yuque/0/2023/png/34598262/1697290649507-237f8572-72b2-4872-9e25-3d6734efde88.png?x-oss-process=image%2Fwatermark%2Ctype_d3F5LW1pY3JvaGVp%2Csize_55%2Ctext_VHV4dW5Eb2M%3D%2Ccolor_FFFFFF%2Cshadow_50%2Ct_80%2Cg_se%2Cx_10%2Cy_10)

关岛常见的东亚语言包括日语、汉语、韩语，甚至俄语。

北马里亚纳群岛也是如此。
