# 介绍

本文档是题库[可学习的俄罗斯+](https://tuxun.fun/map/37712)的补充指南和说明目录。本文档是在由 Illusion 编写的《可学习的俄罗斯》（A Learnable Russia，ALR） 获得巨大成功的基础上推出的。本文档继承ALR的方式和风格，主要目的是填补《可学习的俄罗斯》文档遗留的空白，同时提供更多小众和高级技巧，以帮助玩家更好区域猜测俄罗斯。鉴于 ALR 已经涵盖了大部分独特地区，ALR+中包含的某些地区可能无法 100%与其他地区区分。对于可能与其他地区混淆或可能溢出到相邻联邦主体的地区，我已提供了相关说明。

推荐你在阅读本文档之前先阅读《[可学习的俄罗斯](https://www.yuque.com/chaofun/tuxun/a_learnable_russia_supplementary_guide)》。并且，你应该对俄罗斯的季节和街景车 meta 的原理有基本的了解。如需更详细的分析，请参考 Finbarr 关于俄罗斯季节和街景车 meta 的文档	。

译注：图寻[配套题库](https://tuxun.fun/map/37712)点此链接。本文原作者在编写此文档时很可能假定读者已经很熟悉《可学习的俄罗斯》，和俄罗斯的各种街景车的存在（可以参考前作译者加的附录）。因此强烈建议先学会前作再来学习本作。本文关于天线的翻译有别于《可学习的俄罗斯》，是参照 Plonk It 中的[俄罗斯文档](https://www.yuque.com/chaofun/tuxun/russia)翻译的。如果对文中的天线类型感到疑惑，请点[此链接](https://www.yuque.com/chaofun/tuxun/russia)并翻到第二步的街景覆盖部分。如果对文中内容仍然感到疑惑，或者哪里翻译有误，请加 QQ 2067199168 并说明来意。
